How to Play

1. Select a Tile

Tap any pipe tile (not the green Start or gold Exit) to pick it up. It will glow blue.

2. Move It

Tap an empty cell to place the selected tile there. Rearrange pipes to form a path.

3. Connect Start to Exit

Build a continuous pipe from the green Start block to the gold Exit block.

4. Roll the Ball!

When the path connects, the GO button appears. Press it to watch the ball roll to victory!

Star Rating

Solve in fewer moves for more stars. Use Hints and Undo to help!
