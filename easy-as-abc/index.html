Easy as ABC

Place letters using directional clues

How to Play

  • Tap a cell to cycle through letters (A, B, C, empty)
  • Each row and column must have each letter exactly once
  • Edge clues show the first letter visible from that direction
  • Green cells = correct, Red = conflicting placement
  • Match all clues and fill all letter cells to win
