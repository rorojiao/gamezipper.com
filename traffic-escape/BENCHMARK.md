### Core Mechanic
Click cars to move them in arrow direction. Clear gridlocked intersections by selecting correct order. Each car has fixed directional arrow — order of moves matters (TRUE LOGIC PUZZLE).

### Key Systems
- **Levels**: Dozens + boss levels
- **Level Race**: Compete vs 4 others for fastest completion
- **Scoring**: Time-based race scoring
- **Progression**: Cross-platform save

### Art Style
- Top-down view, colorful cars/roads
- Sharp vibrant graphics
- Realistic sound effects

### Monetization
- Ads after every level (USER COMPLAINT)
- £5.99 to remove (expensive per users)

## UNIQUE DIFFERENTIATOR
Traffic Escape's arrow-based restricted movement creates TRUE LOGIC PUZZLE where order matters.
Other games: free drag (action), timing (skill), match-3 (casual), sliding block (classic).

---

## MUST-HAVE FEATURES FOR GAMEZIPPER
1. Arrow-based movement system (core mechanic)
2. 50+ levels with boss levels every 10
3. Undo button (essential for puzzle genre)
4. Hint system (reveals next correct car)
5. Star rating (based on moves/time)
6. localStorage progress saving with version
7. Various vehicle types (car, truck, bus, emergency)
8. Tutorial (first 3 levels guided)
9. Web Audio BGM + SFX (engine, honk, crash, success)
10. Daily challenge mode for retention

## SHOULD-HAVE FEATURES
1. Level Race mode (competitive timer)
2. Move counter with par system
3. Smooth 60fps animations
4. Particle effects on car exit
5. Progressive difficulty curve
6. Chapter/zone theming

## GAMEZIPPER DIFFERENTIATION
1. Browser-first (most competitors mobile-first)
2. Minimal/rewarded ads only (address #1 user complaint)
3. Canvas-based smooth performance
4. Dark neon GameZipper visual theme
5. Monetag ad integration (zones 110120/110121/110122)
