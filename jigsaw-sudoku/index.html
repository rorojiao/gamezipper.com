Welcome to Jigsaw Sudoku

Instead of standard 3×3 boxes, the grid is divided into irregularly-shaped regions. Each region has the same number of cells as the grid size.

The thick lines show region borders. Every region must contain each digit exactly once.

The Rules

Fill the grid so that every row, every column, and every irregular region contains each digit exactly once.

Selecting a cell highlights its entire region, so you can see which cells share the constraint.
