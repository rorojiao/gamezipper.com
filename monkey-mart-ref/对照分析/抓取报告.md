## 3. 游戏结构分析

### 3.1 游戏引擎
- **引擎类型：** Unity + WebAssembly (WASM)
- **加载器：** dmloader.js (HTML5 Direct Module Loader)
- **核心技术：** Unity WebGL Build + WASM

### 3.2 游戏界面元素（基于截图分析）

**主场景组成：**
- 热带海岛风格背景（蓝天白云、绿色草地、棕榈树）
- MONKEY MART 木制招牌
- 猴子角色（玩家角色、NPC）
- 收银台区域
- 农作物（玉米、香蕉）

**功能区域：**
- 收货区域
- 上架商品区
- 收银台
- 绿色进度条（加载状态）

### 3.3 游戏玩法（参考截图推断）
1. 玩家扮演猴子经营超市
2. 种植/收获农作物（玉米、香蕉）
3. 搬运商品到货架
4. 收银收款
5. 升级系统（从截图可见多种升级选项）
