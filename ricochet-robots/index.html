How to Play

1. Select a robot by tapping/clicking it. The selected robot glows.

2. Choose a direction using the D-pad or arrow keys. The robot slides until it hits a wall or another robot.

3. Reach the target. Guide the matching colored robot to the pulsing target cell.

4. Use other robots as stoppers to reach tricky spots!

Tip: Solve in fewer moves to earn more stars!
