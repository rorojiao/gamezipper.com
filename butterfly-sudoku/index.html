How to Play Butterfly Sudoku

  • Four 9×9 grids overlap in a 2×2 butterfly pattern (15×15 combined grid)
  • Each grid follows standard Sudoku: every row, column, and 3×3 box must contain 1–9
  • The cross-shaped shared region (gold tint) belongs to multiple grids simultaneously
  • Tap a cell, then tap a number (1–9) or use your keyboard
  • Toggle Notes mode for pencil marks • Use Hint if stuck • Check to find errors
