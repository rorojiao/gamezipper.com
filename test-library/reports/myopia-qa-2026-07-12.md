## 🐛 发现并修复的 Bug

### P0 (Critical) — 游戏完全不能玩
**Bug**: `canvas.addEventListener('pointerdown', ...)` 在 script top-level 执行（line 296），此时 `canvas` 变量为 `undefined`（仅在 `init()` 内赋值）。
**症状**: TypeError 中断整个 `<script>`，`init()` 从未执行，canvas 卡在 300×150 默认尺寸，0 像素渲染。游戏 100% 不可玩。
**根因**: 事件监听器注册代码放在了 init() 之外的顶层，但依赖 init() 才赋值的 canvas 全局变量。
**修法**: 包装进 `setupInput()` 函数，在 `init()` 内 canvas 赋值后调用。
**Commit**: `3116279fc2`

### P2 (Medium) — 视觉居中问题
**Bug**: Grid 点阵从 canvas 坐标 (0,0) 开始绘制，但 canvas 尺寸为 (C+1)×(R+1) cells，grid 跨度仅 C×R cells，导致右侧大面积留白、grid 左对齐 + 顶部箭头被裁切。
**修法**: 所有 draw 坐标和 hit-test 坐标统一加 `off = cellSize*0.5` 偏移，grid 居中对称。
**Commit**: `8ae9933138`
