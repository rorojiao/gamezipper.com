### 24.8 WebGPU Three-Domain Memory Leak Detection for Browser Games
- [P-062] **[P2]** WebGPU three-domain memory leak detection (mapped/buffer + texture + pipeline-layout) for browser games — DrCodes (Feb 2026, "Fix WebGPU Memory Leaks in Browser Games: Complete Guide") documents that **WebGPU manages memory across three critical domains** — (a) mapped/buffer memory (`GPUBuffer` via `getMappedRange` / `writeBuffer`), (b) texture memory (`GPUTexture` + `GPUTextureView`), (c) pipeline/binding-layout memory (`GPUPipelineLayout`, `GPUBindGroup`, `GPUSampler`) — and each domain requires domain-specific handling to prevent leaks.
