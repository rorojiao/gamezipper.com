## 2. 核心玩法

### 2.1 一句话描述
在容器中投放水果，相同水果碰撞合成更大的水果，目标是合成最大的西瓜。

### 2.2 游戏循环
1. **开始**: 玩家看到开始界面，点击 Play 开始
2. **进行**: 玩家移动手指/鼠标定位水果位置，点击释放投下水果。两个相同水果碰撞会合成下一级水果，获得分数。连续合成触发连击加分。
3. **结束**: 水果超过顶部危险线超过2秒，游戏结束。显示最终分数和最高分。

### 2.3 核心机制

#### 2.3.1 基本规则
- 11种水果，从小到大：Cherry → Strawberry → Grape → Dekopon → Orange → Apple → Pear → Peach → Pineapple → Melon → Watermelon
- 投放的水果随机从前5种中选择（加权随机）
- 两个相同等级水果碰撞合成下一级
- Watermelon（最高级）不可再合成
- 投放冷却 500ms

#### 2.3.2 得分规则
- 合成得分 = 新水果等级 × 10
- 连击加成：连续合成时，分数 × (1 + combo × 0.3)
- 连击判定：0.8秒内连续合成算连击

#### 2.3.3 失败条件
- 水果顶部超过危险线（y=100）
- 超过危险线持续超过2秒（GAME_OVER_GRACE = 2000ms）
- 注意：刚投放的水果有豁免期（通过冷却时间自然保护）

#### 2.3.4 特殊规则
- 合成大水果（>=7级）时屏幕震动
- 合成 Watermelon 时强烈震动（shakeMag=12）
- 新合成的水果有缩放动画（从1.4倍缩小到1倍）
- 水果有可爱的小眼睛（半径>16px时显示）

### 2.4 输入方式
- [x] 触摸（移动端主要输入方式）
- [x] 鼠标（桌面端备用）
- [ ] 键盘（不适用）
