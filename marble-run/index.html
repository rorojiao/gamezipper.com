How to Play Marble Run

Game Controls

  • Click a piece in the palette to select it
  • Click on the grid to place the selected piece
  • Click an existing piece to remove it
  • Press PLAY to release the marble and test your track
  • Press STOP to halt simulation and edit your track
  • Press RESET to clear all placed pieces

Tips & Tricks

  • Start by tracing a path from START to FINISH, then fill in pieces
  • Use trampolines to reach higher platforms
  • Boosters help the marble maintain speed on long tracks
  • Collect stars by routing the marble through them
  • Experiment - there are often multiple valid solutions

About Marble Run

Marble Run is a physics-based puzzle game where you design marble tracks using ramps, curves, tubes, funnels, trampolines, boosters, and bumpers. Build a path from start to finish while collecting stars for bonus points. With 20 progressively challenging levels, Marble Run tests your spatial reasoning and understanding of physics.

  • Funnels redirect the marble into a narrow path
  • Curves can be rotated by clicking them multiple times
  • The UNDO button removes the last placed piece

Frequently Asked Questions

How do I play Marble Run?

Select track pieces from the palette, then click on the grid to place them. Build a path from the green START zone to the red FINISH zone. Press PLAY to release the marble and test your track.

How many levels does Marble Run have?

Marble Run features 20 hand-crafted levels with progressive difficulty, from tutorial stages to expert multi-section challenges.

What are the stars in Marble Run?

Each level has up to 3 stars scattered on the grid. Guide the marble through stars to collect them. Your star count determines your rating for each level.

Can I undo track placements?

Yes, use the UNDO button to remove the last placed piece. You can also click an existing piece on the grid to remove it.

Is Marble Run free to play?

Yes, Marble Run is completely free to play online in your browser. No download or installation required.
