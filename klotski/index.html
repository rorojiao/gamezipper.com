📖 How to Play

🎯 Goal

Free the 2×2 Cao Cao block (large red block) by sliding it to the bottom-center exit.

🎮 Controls

  • Click/tap a block to select it
  • Drag in any direction to slide it one cell
  • Blocks can only move into empty cells

📦 Block Types

  • ■ 2×2 Cao Cao — the target block to free
  • ▮ 1×2 Vertical — tall blocks (generals)
  • ▬ 2×1 Horizontal — wide blocks
  • ◆ 1×1 Soldier — single-cell blocks

⭐ Stars

Solve in the optimal number of moves for 3★. Fewer moves = better rating!

🏛️ History

Klotski (华容道) is based on a famous scene from Romance of the Three Kingdoms. The classic configuration requires 81 moves — one of the hardest sliding puzzles ever created.
