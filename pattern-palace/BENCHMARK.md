# Pattern Palace — Competitor Benchmark

## Competitor Analysis

### 1. MindArena — Pattern IQ Game
- URL: mindarenagames.com/play/pattern-iq.html
- Core: Spot pattern → complete sequence → test IQ
- Features: Progressive difficulty, visual intelligence puzzles
- Style: Clean, modern

### 2. IQGame.online — Pattern Sequence IQ Game
- URL: iqgame.online/game2-pattern-sequence.html
- Core: Identify missing elements in visual sequences
- Features: Multiple visual puzzle types, logic reasoning training
- Style: Educational, clean

### 3. PatternFind.org — Pattern Recognition Test
- URL: patternfind.org
- Core: Find next shape in sequence, free IQ pattern test
- Features: Online test, visual patterns, answers provided
- Style: Test/assessment format

### 4. YourFaceWhen — Sequence Game
- URL: yourfacewhen.org/games/sequence
- Core: Find the sequence and predict what comes next
- Features: Brain training, pattern recognition
- Style: Casual browser game

### 5. CognitiveTrain — Visual Pattern Test
- URL: cognitivetrain.com/visual-pattern-test
- Core: A → B → C → ? format, series completion
- Features: Inductive reasoning, 3×3 matrix puzzles
- Style: Assessment/training

## Core Systems to Implement
1. **Pattern Engine**: Multiple pattern types (shape, color, size, rotation, count, position)
2. **Difficulty Progression**: Easy (1-2 attributes) → Hard (3-5 attributes simultaneously)
3. **Scoring**: IQ-style scoring, time bonus, streak multipliers
4. **Daily Challenge**: Seeded daily puzzle
5. **Statistics**: Accuracy, avg time, best streak, category breakdown
6. **Tutorial**: Interactive tutorial explaining each pattern type
7. **Hint System**: Reveal one attribute, highlight differences
8. **Achievements**: Speed demon, perfectionist, pattern master, etc.

## Pattern Types (priority order)
1. Shape sequence (circle → square → triangle → ?)
2. Color sequence (red → blue → green → ?)
3. Size progression (small → medium → large → ?)
4. Rotation pattern (0° → 90° → 180° → ?)
5. Count/quantity (1 → 2 → 3 → ?)
6. Fill pattern (empty → half → full → ?)
7. Position movement (left → center → right → ?)
8. Combination of 2+ attributes (shape+color)
9. 3×3 matrix (Raven's Progressive Matrices style)

## Art Style
- Dark neon gradient background (GameZipper standard)
- Bright geometric shapes with glow effects
- Clean, modern UI with glass-morphism panels
- Satisfying animations on correct/incorrect answers
