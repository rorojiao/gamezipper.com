How to Play Pulse Path

1. Read the relay

The number on a relay tells you how many grid cells the pulse travels. A 3-pulse must land exactly three cells away.

2. Choose a direction

Use arrow buttons, keyboard arrows, or swipe from the current relay. Pulses cannot leave the board or pass through blockers.

3. Reach the goal

Every level has one verified route. Decoy relays look tempting, but only the unique route reaches the golden portal.

Tips & Strategy

  • Work backwards from the goal: which relay can land exactly on it?
  • Long pulses are restrictive; solve them before short flexible pulses.
  • If a relay has no outgoing valid pulses, it is probably a decoy.
  • Use hints sparingly — a perfect 3-star solve uses the optimal route.
