Nurimaze

Blacken cells to create a maze path!

  1. Click cells to toggle between black (wall) and white (path).
  2. Cells marked S, G, circle, or triangle cannot be blackened.
  3. No 2x2 area of the same color (black or white).
  4. All white cells must be connected — one single path network.
  5. White paths must be exactly 1 cell wide — no loops allowed.
  6. There must be a path from S to G through all circles, avoiding all triangles.

How to Play

Goal: Create a maze by deciding which cells are black (walls) and which are white (paths).

S = Start, G = Goal

Circle = must be on the S-to-G path

Triangle = must NOT be on the S-to-G path

Controls:

  • Click a cell to apply the current mode (Black/White/Erase)
  • Mode buttons switch between coloring black, white, or erasing
  • Keyboard: 1=Black, 2=White, 3=Erase, H=Hint, R=Restart, Enter=Check, Esc=Menu

Win condition: All rules satisfied and path S→G through all circles avoiding triangles.
