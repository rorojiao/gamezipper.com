🔬 Interferometer Align

A Michelson interferometer splits a laser into two arms. Mirror stages (M1, M2, …) each add a discrete quarter-wavelength path segment — some extend (+), some retract (−).

The interference fringe cycles every λ positions. Adjust each stage's level so the cumulative path difference hits the target fringe → constructive interference.
