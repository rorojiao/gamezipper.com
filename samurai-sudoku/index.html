How to play Samurai Sudoku

  • Goal: Fill all five 9×9 grids so every row, column, and 3×3 box contains digits 1-9 exactly once.
  • Layout: One center grid + four corner grids in a cross. The four corner 3×3 boxes are shared between the center and each corner.
  • Tap a cell then tap a digit (1-9) on the palette, or use keyboard.
  • Notes mode: toggle pencil marks to track candidates per cell.
  • Yellow = given clue, green = your entry, red = conflicting.
  • Complete all five grids to win. Five mistakes = game over.
