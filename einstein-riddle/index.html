    How to Play

    1. Read the clues

    Each clue gives a logical fact about which values go together. There is exactly one arrangement that satisfies all clues.

    2. Click cells to mark

    Click a cell in the grid to cycle through: empty → ✓ (yes) → ✗ (no) → empty. The grid has houses as columns and attributes as rows.

    [ ✓ means: this value IS in this house ]
    [ ✗ means: this value is NOT in this house ]

    3. Use deduction

    Mark every cell you can deduce. When you're confident, click Check Answer. If you make a mistake, the wrong cells flash red.

    4. Hints & stars

    You have 3 hints per puzzle. Each hint auto-fills one correct cell. Fewer hints = more stars!
