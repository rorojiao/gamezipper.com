How to Play

Goal: Place stars so every row, column, and colored region has exactly the required number.

Rule 1: Stars cannot touch — not even diagonally!

Rule 2: Each row must have exactly N stars.

Rule 3: Each column must have exactly N stars.

Rule 4: Each outlined region must have exactly N stars.

Controls: Tap/click a cell to place or remove a star. Long-press to mark as empty (X).
