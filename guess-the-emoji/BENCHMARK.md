# Guess The Emoji — Competitive Benchmark

## Game Overview
Players guess words/phrases from emoji combinations. Emoji puzzles display 2-5 emojis that represent a word, movie title, song, celebrity, or common phrase. Players type their answer in a text input.

## Top Competitors

### 1. Guess The Emoji (Conversion LLC / Random Logic Games)
- **Platform**: iOS + Android (100M+ downloads combined)
- **Levels**: 1000+ puzzles across 40+ categories
- **Categories**: Movies, TV Shows, Characters, Places, Food, Animals, Sports, Music, Brands, Everyday Phrases, Idioms
- **Monetization**: Interstitial ads every 5 levels, banner ads, hint IAP
- **Key Features**:
  - Reveal letter hint (costs coins)
  - Remove letters hint (costs more coins)
  - Free coins from daily bonus + completing categories
  - Facebook share to get hints
  - Category locked until previous completed
  - Streak bonus for consecutive correct answers
- **UI Style**: Clean, colorful, emoji-centered. Letter tiles at bottom.

### 2. Emoji Quiz (Apprope)
- **Platform**: iOS + Android (50M+ downloads)
- **Levels**: 500+ puzzles
- **Categories**: Similar to Guess The Emoji
- **Key Features**: 
  - Multiple choice (easier mode) + free text (harder mode)
  - Hint system with coins
  - Skip level option
- **UI Style**: Emoji large at top, blank tiles for answer below

### 3. Poke the Emoji (Game maker on Poki)
- **Platform**: Web (Poki)
- **Key Features**: Screenshot-based emoji identification

### 4. Emoji Puzzle (Web variants on CrazyGames)
- **Platform**: Web browser
- **Simple**: Show emojis, type answer or select from options

## Competitive Analysis Summary

### Must-Have Features
1. **200+ emoji puzzles** across categories
2. **Category system** (Movies, TV, Food, Animals, Sports, Music, Brands, Phrases, Idioms, Countries, Professions)
3. **Hint system** (reveal letter, remove extra letters)
4. **Coin/reward system** (earn coins for correct answers, spend on hints)
5. **Daily puzzle** (new puzzle each day for retention)
6. **Letter tile display** (blank tiles showing answer length, fill in letters)
7. **Progress tracking** (localStorage, category completion %)
8. **Streak tracking** (consecutive correct without hint)
9. **Share results** (copy emoji + score to clipboard)

### Differentiators
- **Reveal word hint** as last resort (more expensive)
- **Difficulty tiers** per category (Easy/Medium/Hard based on obscurity)
- **Achievement system** (complete all in a category, 10-streak, etc.)
