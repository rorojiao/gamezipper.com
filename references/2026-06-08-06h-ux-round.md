### Detection Recipe (for future rounds)
```python
import re
c = open('index.html').read()
static = set(re.findall(r'rel="prefetch"[^>]*href="(/[^"]+)"', c))
m = re.search(r"topGames=\[(.*?)\]", c)
js = set(re.findall(r"'(/[^']+)'", m.group(1)))
if static != js: print("PREFETCH DESYNC")
```

## P1: WCAG 2.2 Focus Not Obscured (scroll-margin on game-card)

### Issue
`.game-card` had no `scroll-margin-top`, so when navigating with keyboard (Tab), the focused card could be hidden behind the sticky header (80px height on desktop, ~60px on mobile).

### Fix
Added to the existing `scroll-margin-top` rule at L373:
```css
.game-card{scroll-margin-top:88px}
@media(max-width:600px){.game-card{scroll-margin-top:68px}}
```
88px = 80px sticky header + 8px visual breathing room. Mobile: 68px = 60px header + 8px.
