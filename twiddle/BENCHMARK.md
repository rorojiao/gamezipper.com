# Twiddle — Competitive Benchmark

## Game Type
Tile-rearrangement puzzle from Simon Tatham's Portable Puzzle Collection.

## Rules (Canonical — verified via official Tatham docs)
- Grid of numbered tiles, goal: arrange in ascending order (1, 2, 3, ... N)
- Basic move: rotate a 2x2 square of tiles clockwise or anticlockwise
- Advanced: rotate 3x3 (or larger odd) square blocks
- Orientation mode (optional): each tile has a triangle; all must point up to win
- Every scrambled state is solvable (puzzle generated by reversing actual rotations)

## Systems
- Level select with lock + star ratings (3-star based on move efficiency)
- Timer + best time per level (localStorage)
- Undo (full move history)
- Reset (restart current level)
- Hint (highlight a useful rotation)
- Tutorial overlay (3-step onboarding)
- Sound effects (Web Audio API): rotate, win, star, button
- Procedural BGM (Web Audio API ambient loop)
- Mute toggle (SFX + BGM separate)
- Progress save (localStorage v1): unlocked levels, stars, best times
- Particle celebration on win
- Mobile touch support (pointer events)
