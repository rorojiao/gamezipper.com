### Core Mechanics: 4-Station Workflow

1. **Order Station** — Accept customer orders, manage queue, patience timer
2. **Build Station** — Select cup size, pour ice cream, add syrups/blends
3. **Mix Station** — Blend with timing indicator (green zone = perfect)
4. **Top Station** — Add whipped cream, cherries, sprinkles (precision placement)

### Scoring System
- Star rating per order (Bronze/Silver/Gold) based on accuracy + speed
- Tip economy: better orders = more tips
- Tips reinvested into shop upgrades

### Progression
- Day-based levels with increasing difficulty
- New customer types with unique orders/preferences
- Shop upgrades (equipment speed, decor, clothing)
- Unlockable ingredients/toppings

### Why These Games Are Addictive
1. Low barrier, high ceiling — easy to learn, hard to master
2. No randomness — skill determines outcomes
3. Clear feedback — immediate visual/audio response
4. Collection & unlocking drives long-term engagement
5. Time pressure creates exciting flow state

### Differentiation Opportunity for GameZipper
- Modern HTML5/JS (no Flash dependency)
- Responsive, touch-optimized for mobile web
- Clean modern visual style (dark neon GameZipper theme)
- Quick play sessions optimized for casual browser play
