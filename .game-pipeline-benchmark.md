### Core Mechanic Analysis

**Gameplay Loop:**
1. Tap connected group of 2+ same-colored cubes → they blast
2. Cubes above fall down (gravity)
3. New cubes spawn from top to refill grid
4. Clear cubes to fulfill level objectives within move limit
5. Earn stars based on moves remaining

**Grid Settings:**
- Size: 8x8 or 9x9
- Colors: 5-6 (red, blue, green, yellow, purple, orange)
- Initial fill: random distribution

**Special Cubes (created by clearing large groups):**
| Group Size | Special | Effect |
|------------|---------|--------|
| 5-6 | Rocket | Clears entire row or column |
| 7-8 | Bomb | Clears 3x3 area |
| 9+ | Disco Ball | Clears all cubes of one color |

**Obstacles:**
- Crate: Takes 2 adjacent clears to destroy
- Ice: Frozen cube inside, clear adjacent to break
- Balloon: Floats up, must clear all before it reaches top
- Duck: Must be brought to bottom by clearing below it
- Stone: Immovable, blocks gravity and clears

**Level Objective Types:**
1. Collect X cubes of specific colors (e.g., "15 red, 10 blue")
2. Clear Y obstacles (e.g., "Clear 10 crates")
3. Save N items (e.g., "Save 5 ducks")
4. Reach target score
5. Mixed objectives (combine multiple types)

**Move Limits:**
- Early levels: 15-20 moves
- Mid levels: 20-25 moves
- Late levels: 25-30 moves

**Power-ups (consumed per level):**
- Hammer: Tap any cube to remove it (3 charges)
- Shuffle: Randomize all cubes (1 charge)
- Extra Moves: +5 moves (1 charge)
- Bomb: Place a bomb to clear 3x3 (limited)

**Monetization:**
- Coins earned from levels
- Spend coins on power-ups
- Optional IAP for coin bundles
- Rewarded ads for bonus moves/coins
- Lives system (5 lives, regenerate over time)

### Key Success Factors

1. **Juicy Feedback:** Cube pop animations, particles, screen shake, sound effects
2. **Clear Objectives:** Players always know what to do next
3. **Skill Expression:** Efficient group selection leads to higher scores
4. **Progression:** 100+ levels with increasing complexity
5. **Power-up Strategy:** Strategic use of limited resources
6. **Visual Polish:** Bright colors, cute graphics, satisfying animations

### What to Implement

Core: Grid + tap groups + gravity + objectives + moves + stars
Special: Rocket, Bomb (skip Disco Ball for MVP)
Obstacles: Crates only for MVP
Power-ups: Hammer (simplest)
Levels: 30 levels across 5 tiers (6 levels per tier)
Audio: Pop SFX + BGM loop
UI: Level select + objectives + move counter + hammer button

### Gap vs Same Game

| Feature | Same Game | Cube Pop | Difference |
|---------|-----------|----------|------------|
| Game structure | Endless | Level-based | Clear goals vs infinite play |
| Objectives | Clear board | Collect/clear/save | Varied goals |
| Move limit | None | Limited | Strategic depth |
| Special blocks | None | Rocket/Bomb | Combo satisfaction |
| Obstacles | None | Crates | Puzzling |
| Refill | None | Yes | Continuous play |
| Progression | High score | Unlock levels | Achievement |

Conclusion: While both games share the tap-to-clear base mechanic, Cube Pop is a fundamentally different game with level progression, objectives, strategic resource management, and special effects. Same Game is a score-chasing game; Cube Pop is a puzzle-solving game.
