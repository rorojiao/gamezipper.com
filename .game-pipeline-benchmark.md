# Competitive Benchmark — Balls vs Bricks (Round 41)

## Top competitors (mobile, 100M+ download tier)
1. **Bricks Breaker** (5con / PlayDots) — 100M+ downloads. THE category leader.
   - Aim direction with finger drag → release to fire a stream of balls
   - Bricks have integer HP; each ball hit decrements HP by 1
   - Collect +1 ball icons mid-level to grow your ball count (snowball loop)
   - Bricks descend one row after each shot; game over if they reach the bottom
   - Endless mode + level mode + boss bricks
   - Powerups: laser, bomb, split-ball
2. **Bricks n Balls** (Cheetah Games) — 100M+. Similar core + themed worlds.
3. **Ballz** (Ketchapp) — 50M+. Minimalist aesthetic, endless-focused.
4. **Balls vs Bricks Breaking Game** — 50M+. Variant with brick layouts.

## Core mechanics to match
- **Aim & shoot**: drag from launcher to set angle, dotted trajectory preview line
- **Ball stream**: N balls fire sequentially with small spacing, all same angle
- **Bounce physics**: balls reflect off left/right/top walls; vanish at bottom
- **Brick HP**: each brick shows a number; -1 per ball hit; clears at 0
- **+1 ball powerups**: floating icons; collect by hitting with a ball → ball count grows
- **Descending rows**: after every shot, all bricks shift down one row; new row spawns at top
- **Win/lose**: clear all bricks = win; any brick reaches the bottom row = lose

## Special brick types (Tier 3+)
- **Indestructible** (gray, ∞ HP) — obstacle, must path around
- **Bomb brick** (red, ×3) — explodes 3×3 area when destroyed
- **Splitter brick** — on destruction, spawns 2 extra balls from its position
