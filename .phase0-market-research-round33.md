### Selected: PathPix (slug: `pathpix`)

**Mechanic**: Numbered-pair path-drawing puzzle where each pair of identical
numbers specifies an EXACT path length to connect them. Draw a path of exactly
N cells between each pair. When all pairs are connected with their exact-length
paths, every cell is filled — and the filled paths collectively REVEAL a hidden
pixel-art picture.

**This is mechanically DISTINCT from the existing connect-route cluster** (11
games: numberlink, flow-connect, path-finder, one-line-connect, pipe-connect,
etc.):

1. **Exact-length constraint**: In Numberlink/Flow Free, you connect pairs with
   ANY-length path (constraint: don't cross, fill the grid). In PathPix, each
   pair's number IS the required path length — the path must be exactly N cells
   long, no more, no less. This completely changes the solving logic: you must
   route paths to hit an exact cell count, not just reach the endpoint.

2. **Picture-reveal payoff**: The completed paths form a recognizable pixel-art
   image (animals, objects, scenes). This is the defining PathPix experience —
   solving a logic puzzle to discover a hidden picture. No existing connect-
   route game in the catalog has a picture-reveal mechanic.

3. **Standalone number-as-length (not number-as-label)**: In Numberlink, the
   number on a pair is just a label/match-ID. In PathPix, the number is the
   LENGTH CONSTRAINT — it actively constrains the path geometry. This is a
   fundamentally different rule set.

The connect-route saturation cluster applies to "route a connection between
nodes" verbs. PathPix's verb is "draw exact-length paths that compose a hidden
image" — the length constraint + image reveal are the distinct non-routing
mechanics, analogous to how Klotski (2×2 main block) was distinct from parking-
jam (1×N cars) despite both being "slide blocks on a grid."

### Why PathPix over the other candidates
1. Hexcells: hex-minesweeper already covers the hex-grid-logic-minesweeper
   verb. Hexcells adds {n} + continuous/non-continuous bracket clues, but the
   CORE verb (deduce mines from adjacency numbers on a hex grid) is identical.
   14/25 — below gate.
2. All Nikoli candidates (Hitori, Heyawake, Fillomino, Ripple Effect, Light Up,
   Yajilin, Masyu, Slitherlink, Nurikabe, Tents) are ALREADY LIVE — catalog has
   40+ Nikoli types. The Nikoli family is comprehensively covered.
3. Cryptograms, Knotwords, Boggle: all already shipped.
4. PathPix is a NAMED, RECOGNIZED franchise (15+ years, PathPix Pro 320
   puzzles, kpixgames) — name recognition drives search. The image-reveal
   mechanic is genuinely absent from the 532-game catalog.

### Decision
**Build PathPix** as game #533. Apply constructive level generation:
1. Design a pixel-art target image (simple shapes: cat, heart, star, house,
   tree, fish, etc.) on a grid.
2. Decompose the image into path segments: each contiguous run of same-color
   cells becomes a pair (two endpoint numbers), with the path length = number
   of cells in that segment.
3. Place the numbered pairs on the grid; the solver draws exact-length paths
   to reconstruct the image.
4. Verify unique solvability via backtracking solver (path must be exactly N
   cells, no crossing, all cells filled).

30 levels across 5 tiers (Beginner 5×5 → Master 12×12). Full S-tier: levels +
art + music + QA.
