How to Play

Goal: Paint cells black or leave them white to match all row and column clues.

  • Numbers outside the grid show how many cells should be black in that row/column.
  • Click or tap a cell to toggle it between white and black.
  • Use deduction: if a row clue is 0, leave it all white.
  • Cross-reference rows and columns to solve each puzzle.
  • Each puzzle has exactly one unique solution!

Tips: Start with rows/columns that have 0 or maximum clues. Use hints if stuck.
