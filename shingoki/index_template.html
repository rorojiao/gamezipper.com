🧩 How to Play Shingoki

Goal: Draw a single continuous loop that passes through ALL circles.

⚫ Black circles: The loop must TURN (90°) at these cells.

⚪ White circles: The loop must go STRAIGHT through these cells.

🔢 Numbers: Each number tells you the length of the straight segment that includes the circle.

Rules: The loop runs horizontally/vertically, never crosses itself, and forms a single closed path.

Controls: Drag between adjacent cells to draw path segments. Use Erase mode to remove segments.

Keyboard: 1=Draw, 2=Erase, H=Hint, R=Clear, Enter=Check, Esc=Menu
