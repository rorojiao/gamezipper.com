## Paint Mechanic (CRITICAL — read carefully)
1. Flag renders with ALL regions EMPTY (white #F5F5F5 fill + dashed gray outline) EXCEPT background which starts empty too
2. Color palette at bottom shows the flag's unique colors as large tappable buttons (min 56px)
3. Player taps a color → it highlights (glow ring)
4. Player taps a region → it PAINTS with the selected color via radial-spread animation (animate a growing circle clip from tap point over 300ms)
5. Auto-win check after each paint: if every region's currentColor === its target color → level complete
6. Beginner/Intermediate (levels 1-20): wrong colors can be re-painted freely (tap again with different color). NO hearts. Relaxing.
7. Advanced/Expert (levels 21-30): once a region is painted with a WRONG color, it LOCKS (can't repaint) and costs 1 heart. 0 hearts = level fail → retry. Correct colors also lock (with green glow).
8. Hint button: auto-fills one random empty region with its correct color. 3 hints per level.

## Region Hit-Testing
- Render regions back-to-front (bg first, then overlays)
- Hit-test FRONT-to-back (topmost region under pointer wins)
- For stripes: each stripe is its own region
- For crosses: render as union of vertical bar + horizontal bar regions, hit-test as combined
- Background fills any area not covered by overlay regions
- Canvas pixel-based hit testing acceptable: render each region to an offscreen mask, check pixel under tap
