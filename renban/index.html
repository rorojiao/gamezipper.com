How to Play Renban

Goal: Fill every cell with a number from 1 to N.

Rows: Each number appears exactly once per row.

Columns: Each number appears exactly once per column.

Regions: Cells in the same outlined region must contain consecutive numbers (like 3-4-5 or 5-3-4, not 3-5-6).

Controls: Click a cell, then tap a number. Use Erase to clear. Toggle Notes for pencil marks.
