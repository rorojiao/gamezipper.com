🎯 How to Play

  • Tap a cell to select it (it glows gold), then tap an adjacent cell to join them into a domino.
  • Tap any cell of an existing domino to split it apart.
  • Every domino pair (like 2-5) may appear only once in the whole grid.
  • Use all dominoes in the set exactly once to win. The tracker shows which are placed.
  • Red highlight = duplicate domino. Each puzzle has exactly one solution.
