# Spot the Difference — Competitive Benchmark

## Top Competitors

### 1. spotdiffs.com
- 100+ levels, easy/medium/hard modes
- Daily puzzles, no signup required
- Side-by-side image comparison

### 2. spotdifferent.com
- 100+ levels, clean UI
- No timer on easy modes
- Hint system, zoom feature

### 3. Find The Difference (CrazyGames)
- 10 differences per level
- Various picture themes
- Observation skills focus

### 4. Spot the Differences (Poki)
- Fairy tale scenes, food, street views
- Cute cartoon art style
- Multiple categories

### 5. Differences by Easybrain (Mobile)
- No timer (relaxing mode)
- Zoom for tiny details
- Progressive difficulty
- Hints system

## Must-Have Systems

1. **Level System**: 30+ levels with procedural generation, multiple categories (nature, city, animals, food, fantasy)
2. **Difference Detection**: 5-7 differences per level, tap to circle/mark
3. **Hint System**: 3 hints per level, reveals one difference location with highlight pulse
4. **Timer**: Optional countdown timer per level, bonus points for speed
5. **Zoom**: Pinch-to-zoom on mobile, scroll zoom on desktop
6. **Score System**: Points per difference found, time bonus, streak multiplier, star rating (1-3 stars)
7. **Progress Save**: localStorage with version, track completed levels, best scores, stars
8. **Tutorial**: First level as interactive tutorial
9. **Categories/Themes**: At least 6 visual themes
10. **Sound Effects**: Found difference, wrong tap, hint use, level complete, time warning
11. **Animations**: Circle reveal on correct tap, shake on wrong tap, confetti on level complete
12. **Difficulty Curve**: Early levels = 3-4 easy differences, later = 6-7 subtle differences, smaller differences
13. **Level Select**: Grid view with stars, locked/unlocked states
14. **Streak Tracking**: Consecutive levels without hints
15. **Daily Challenge**: One special level per day

## Visual Style
- Clean, colorful procedural illustrations (Canvas-drawn scenes)
- Side-by-side layout on desktop, stacked/scroll on mobile
- Neon accent for found differences, subtle animations
- GameZipper dark theme with gradient backgrounds

## Core Gameplay
- Two images shown side by side (or stacked on mobile)
- Player taps on differences to circle them
- Wrong taps: brief penalty (time loss or shake feedback)
- All differences found → level complete with star rating
- Progress bar showing X/N differences found
