# Signpost — Competitor Benchmark

## Game
**Signpost** (aka *Pfeilpfad* / "Arrow Path") — canonical logic puzzle from
Simon Tatham's Portable Puzzle Collection (contributed by James Harvey; rules
credit to Janko.at). GameZipper game **#419**, slug `signpost`.

## Rules (authoritative — Tatham doc, confirmed)
- A grid of cells. Every cell except the LAST contains an 8-direction arrow
  (N, NE, E, SE, S, SW, W, NW).
- Number the cells 1..N so that the cell holding **k** points (via its arrow)
  to the cell holding **k+1**. The next cell may be **any distance** along the
  arrow's ray (same row / column / diagonal, correct side).
- The **first** (1) and **last** (N) numbers are pre-revealed; some interim
  numbers may be revealed as starting clues.
- Goal: connect every cell into one continuous chain 1 → 2 → … → N.

## Why selected (Phase 0)
GZ has **418 games** saturating virtually every classic mechanic (sokoban,
mastermind, lights-out, tower-of-hanoi, sliding, tangram, peg-solitaire,
klotski, pipes, all Nikoli types, all Sudoku variants). **Signpost is genuinely
uncovered**: distinct from Numbrix (orthogonal, no arrows, adjacent steps),
Numberlink (pair-connect), Masyu (loop), Hashi (bridges). It is a directional
number-path puzzle — a fresh mechanic. Generator pattern is proven (Hamiltonian
path → derive arrows → uniqueness solver), identical in spirit to the shipped
Numbrix (#411).

## Level data integrity
- 27 levels generated by `gen.py` (Hamiltonian 8-dir path + arrow derivation +
  gap-based givens + uniqueness solver).
- **Independent Node.js verifier (`verify_independent.js`): 27/27 UNIQUE +
  VALID** (structural checks + independent solution counter).
- Givens always include cell 1 (start) and cell N (end).
- Difficulty: Beginner gap=3 (5×5, 9 givens) → Master gap=6 (7×7, 9 givens/49).

## Scoring formula
- Time-based 3-star: bronze/silver/gold thresholds scale with grid size.
- Move counter: each link = 1 move; bonus for solving near optimal link count.
- Best time + best stars persisted per level (localStorage, version 1).

## Visual style
- GameZipper dark-gradient theme, neon accent (amber/gold arrows on indigo).
- Arrows rendered as crisp directional glyphs; given numbers in gold, player
  numbers in white; completed chain cells glow.
- Smooth link-draw animation between connected cells; completion celebration
  (particles + pulse).
