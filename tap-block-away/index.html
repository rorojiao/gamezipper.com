Tap Block Away — Free Online 3D Block Removal Puzzle

Play Tap Block Away free online — the addictive 3D block removal puzzle! Tap blocks with directional arrows to slide them off the grid. Find the right removal order across 30 levels and 5 tiers. Mobile-friendly, no download.

How to Play

  • Each block has an arrow showing the direction it can slide.
  • Tap a block to slide it off the grid in its arrow's direction.
  • A block can only move if nothing blocks its path in that direction.
  • Remove all blocks from the grid to complete the level.
  • Later levels add 3D depth — blocks can be stacked in multiple layers.
  • Tap and rotate the view to see blocks hidden behind others.

Tips & Tricks

  • Start from the edges: Blocks on the outer edges with clear exit paths are safe first moves.
  • Follow the arrows: A block's arrow tells you its only exit direction — plan around it.
  • Work backwards: Think about which block must be removed last, then reason backwards.
  • Rotate often: In 3D levels, rotate the view frequently to spot blocks hidden behind others.
  • Avoid deadlocks: If removing a block traps another, undo and try a different order.
