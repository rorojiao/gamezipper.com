Build a complete, commercial-quality HTML5 logic puzzle game called "Family Tree" — a deductive genealogy puzzle where players drag character tiles into a family tree based on text clues.

## GAME CONCEPT
This is a clone of the hit mobile game "Family Tree! - Logic Puzzles" by Lion Studios (10M+ downloads, 4.6 stars). Players reconstruct a family tree by reading clues and placing character portrait tiles into the correct slots. 3 hearts — wrong placement loses one. Win when all slots filled correctly.

## CORE MECHANIC
1. A family tree diagram renders with some nodes pre-filled (anchors) and several EMPTY SLOTS
2. Character portrait tiles appear in a scrollable tray at the bottom
3. Text clues are shown in a panel
4. Player DRAGS a tile from the tray and DROPSS it onto an empty slot
5. CORRECT placement → tile locks in with a satisfying glow/bounce animation + success sound
6. WRONG placement → screen shakes, lose 1 heart, error sound, tile returns to tray
7. Lose all 3 hearts → game over → restart level
8. Fill ALL slots correctly → level complete → confetti celebration + stars earned

## TREE STRUCTURE
- Nodes arranged in generations (top = grandparents, middle = parents, bottom = children)
- Married couples drawn side-by-side connected by a horizontal "marriage" line
- Parent→child connections drawn as vertical/diagonal lines
- Siblings share parent nodes
- Each node is a rounded card showing: portrait (emoji avatar), name, optional profession tag
- Empty slots show a dashed outline silhouette
- Blue accent = male, pink accent = female (helps with gender clues)

## CLUE TYPES (implement generators for ALL of these)
1. DIRECT: "Mary is John's daughter." / "Robert is the father of Lisa."
2. MARRIAGE: "Quincy is married to Dorsey." / "Tom is married to the nurse." (profession)
3. SIBLING: "Alex is the only child of Sarah." / "Willie has 2 older sisters."
4. AGE: "Joe is older than Missy." / "Ben is the older twin."
5. EXCLUSION: "James is an only child." / "Kate is not married to Paul."
6. STEP-FAMILY: "Zachary's step-father is Randy."
7. CHAINED: "John's only brother is the father of Sarah." (hardest)
