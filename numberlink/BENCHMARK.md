# Numberlink — Competitive Benchmark

## Top 3 Competitors

### 1. Flow Free (Big Duck Games) — 100M+ downloads
- Grid: 5×5 to 15×15, 2500+ free puzzles
- Scoring: Stars for perfect (fewest moves)
- Features: Hints (IAP), undo, daily puzzle, time trial, color themes
- Monetization: Free+ads, $1.99-2.49/pack, $4.99 unlock all
- Art: Clean flat colorful pipes on white grid

### 2. Puzzle Baron Numberlinks — 50,000+ puzzles
- Web-based, timer + monthly leaderboard
- Features: Resume (24h), competitions, Hall of Fame
- Monetization: Free+premium ad removal

### 3. eqsy.io — Daily Numberlink
- Daily 11×11 challenge, sophisticated scoring formula
- Features: Hints (with penalty), undo, daily leaderboard
- Art: Modern clean web design

## Our Implementation Must Include
- Grid sizes: 5×5 (easy) → 11×11 (hard), 25+ levels
- Scoring: Move count + time → star rating
- Systems: Undo, hints (limited), daily puzzle, progress save (localStorage)
- Tutorial: Interactive first-puzzle walkthrough
- Sound: Web Audio procedural SFX + ambient BGM
- Art: Dark gradient GameZipper style, neon-colored paths
- All cells must be filled (Nikoli standard)
- Dark mode by default (differentiator!)
