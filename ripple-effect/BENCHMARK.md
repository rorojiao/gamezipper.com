# Ripple Effect — Competitive Benchmark

## Competitors
1. **Ripple Effect Puzzle** (Frenchie Games Inc, Android) — 1K+ downloads, minimalist, daily puzzles, hints, relaxing audio
2. **Ripple Effect - Logic Puzzle** (Stanley Lam, Android) — 5 sizes × 5 difficulties, procedural generation
3. **Nikoli** (Print/Web) — Original publisher, handcrafted puzzles

## Key Features to Implement
- **Scoring**: Points per cell filled, time bonus, combo multiplier for streaks
- **Levels**: 25+ levels, progressive difficulty (6×6 → 8×8 → 10×10 → 12×12)
- **Hint System**: 3 hint types (reveal cell, check errors, show possibilities)
- **Tutorial**: Interactive step-by-step with animations
- **Timer**: Optional, with time bonus scoring
- **Undo/Redo**: Full move history
- **Progress**: localStorage with version field
- **Daily Challenge**: Seeded daily puzzle
- **Statistics**: Games played, best times, win streak
- **Achievements**: First win, speed run, no hints, etc.
- **Sound**: Web Audio procedural — satisfying placement sounds, error buzz, completion fanfare, ambient BGM

## Visual Style Reference
- Dark gradient background (GameZipper style)
- Neon accent colors (cyan, purple, pink)
- Glass-morphism panels
- Smooth animations on number placement
- Region highlighting on hover
- Error highlighting with red glow
- Celebration particles on level completion

## Music Style
- Ambient electronic, mysterious, ethereal, slow tempo
- Soft synthesizer pads, gentle piano
