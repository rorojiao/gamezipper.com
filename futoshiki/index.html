How to Play Futoshiki
Goal: Fill the grid so each row and each column contains every digit 1..N exactly once, and every inequality sign is satisfied.
Inequality signs: The open side of < or > points to the larger number.
5 > 2    3 < 7
If two cells show ▢ > ▢, the digit on the left must be bigger.
Controls: Tap an empty cell to select it, then tap a number to place it. Tap the same number again to erase it. Toggle Notes to pencil in candidates.
Tips: Every puzzle has exactly one solution, so you never need to guess. Look for rows or columns that are almost full, and inequality chains that force an ordering.
