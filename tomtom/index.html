TomTom
Arithmetic grid puzzle. Fill the grid so each row and column has digits 1-N. Match cage targets!
Settings
Level Complete!
How to Play
Rules:
1. Fill each row and column with digits 1 to N (no repeats).
2. The grid is divided into cages with a target number and operator.
3. Digits in a cage must produce the target when the operator is applied.
4. For - and / cages, any valid ordering works (e.g. 5-3=2 or 3-5=-2 if target is 2).
5. Single-cell cages show the exact digit.

Tip: Unlike KenKen, digits CAN repeat within a cage in TomTom!
