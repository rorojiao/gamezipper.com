# Phase 1: 候选选择 - Pull the Pin Puzzle

## 选定候选
- **游戏名**: Pull the Pin Puzzle
- **Slug**: `pull-the-pin-puzzle`
- **优先级**: 🥇 第一优先 (评分 22/25)
- **选择时间**: 2026-06-28

## 选择理由

### 市场验证
- Poki 有 2+ 个拉线变体 (Pull the String, How to loot)
- 拉线解谜在移动端持续热门
- "pull the pin" 搜索热度高

### 技术可行性
- Matter.js 物理引擎已集成
- Canvas 渲染成熟
- 程序化关卡生成可行

### 竞争优势
- GameZipper 无拉线类游戏
- 可以差异化设计 (救星星、多关卡、成就系统)

### SEO 价值
- 高搜索量关键词
- 长尾关键词丰富

## 游戏规格

### 基础信息
- **类别**: puzzle
- **标签**: pull-pin, physics, puzzle, casual, mobile, brain, gravity, save, logic
- **游戏数**: 30-50 关卡

### 核心玩法
1. 拖拔销钉释放物体
2. 利用重力解决问题
3. 避开危险 (炸弹、尖刺)
4. 收集星星 (1-3星评级)

### 技术规格
- **物理引擎**: Matter.js
- **渲染**: Canvas 2D
- **交互**: 触摸拖拽/鼠标
- **关卡生成**: 程序化 (难度递增)

### 元素列表
- 🟣 主角 (球体角色)
- 🔴 炸弹 (危险)
- ⭐ 星星 (收集品)
- 📌 销钉 (可拉)
- 🧱 障碍 (不可穿)
- 🎯 目标区域 (到达处)

## 关卡设计思路

### 初期关卡 (1-10)
- 单销钉,简单重力
- 教学引导
- 无危险元素

### 中期关卡 (11-30)
- 多销钉,顺序要求
- 引入危险元素
- 简单逻辑推理

### 高级关卡 (31-50)
- 复杂销钉网络
- 时间限制 (炸弹倒计时)
- 多解性 (不同拉法)
