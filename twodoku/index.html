How to play Twodoku

  • Goal: Fill both 9×9 grids so every row, column, and 3×3 box contains digits 1-9 exactly once.
  • Layout: Two grids sit side-by-side. The middle 3×3 stack (highlighted amber) is shared — a digit there counts for both grids.
  • Tap a cell then tap a digit (1-9) on the palette, or use keyboard.
  • Notes mode: toggle pencil marks to track candidates per cell.
  • Yellow = given clue, green = your entry, red = conflicting.
  • Complete both grids to win. Three mistakes = game over.
