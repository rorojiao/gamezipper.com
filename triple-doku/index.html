How to play Triple Doku

  • Goal: Fill all three 9×9 grids so every row, column, and 3×3 box contains digits 1-9 exactly once.
  • Layout: Three grids sit side-by-side in a 9×21 playfield. The amber 3×3 stacks are shared — a digit there counts for both adjacent grids.
  • Grid A (teal border) ← shared AB → Grid B (cyan border) ← shared BC → Grid C (green border).
  • Tap a cell then tap a digit (1-9) on the palette, or use the keyboard.
  • Notes mode: toggle pencil marks to track candidates per cell.
  • Cream = given clue, cyan = your entry, red = conflicting.
  • Complete all three grids to win. Three mistakes = game over.
