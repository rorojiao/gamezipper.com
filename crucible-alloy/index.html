How to Play Crucible Alloy

Crucible Alloy is a metallurgy puzzle where you blend metal ingots in a crucible to match a target alloy specification. Each ingot you add shifts three properties — density, luster, and temper — by fixed amounts (shown above each ingot as +X/+Y/+Z). The goal is to get all three properties to simultaneously hit zero (the amber target marker), which means the alloy is perfectly balanced. The properties operate in modular arithmetic (mod N), so adding too much of one metal wraps the value around.

Controls: Tap (or click) a metal ingot at the bottom to add one unit of it to the crucible. On mobile, long-press (hold for ~0.5s) to remove an ingot; on desktop, right-click to remove. The three property bars show current values vs. target (0). Use ◀ Prev / Next ▶ to navigate levels, ↻ Reset to clear the crucible, and 🔊 to mute.

Tips & Strategy

  • Read the +X/+Y/+Z dials: Each ingot shows how it shifts density, luster, and temper. Plan combinations that cancel out unwanted shifts — you need all three to reach 0 simultaneously.
  • Modular arithmetic wraps: If a property is at N−1 and you add +2, it wraps to 1, not N+1. Use this to your advantage when close to the target.
  • Right-click / long-press to remove: If you overshoot, remove ingots rather than resetting — it's faster and helps you learn which combinations work.
  • Match the "Optimal" count: Each level shows the optimal number of ingots. Fewer ingots = better, though only matching all three properties to 0 completes the level.
  • Metals escalate: Early levels use Copper & Tin (2 metals), mid-levels add Zinc (3), then Iron/Cobalt/Nickel/Manganese (4), and late levels use Gold/Platinum/Palladium/Iridium/Osmium (5 precious metals).
