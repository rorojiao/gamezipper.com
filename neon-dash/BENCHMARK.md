# Neon Dash — Competitive Benchmark

## Game Concept
Geometry Dash-style rhythm platformer: one-tap control, side-scrolling, obstacle-filled levels synced to music beats.

## Competitors Analyzed
1. **Geometry Dash** (RobTop Games) — 500M+ plays, #1 FastPlayGames
   - Game modes: Cube (jump), Ship (fly), Ball (gravity flip), Wave (zigzag), UFO (tap fly)
   - Level system: Practice mode with checkpoints, Normal mode
   - Features: Custom icons/skins, music sync, star ratings, demon levels
   - Monetization: Premium game on mobile, free on web clones

2. **Geometry Dash Lite** — Browser version
   - One-tap controls, instant restart
   - Multiple difficulty tiers
   - Audio-visual sync

3. **The Impossible Game** — Similar genre
   - One-button platformer
   - Practice mode with flags
   - Level editor

## Systems to Implement
- [x] One-tap/click control (tap to jump, hold for ship mode)
- [x] Multiple game modes: Cube (jump), Ship (fly), Ball (gravity flip)
- [x] 30+ levels with progressive difficulty
- [x] Practice mode with checkpoints
- [x] Music-synced obstacles
- [x] Custom character skins/colors (unlockable)
- [x] Star rating per level
- [x] Progress saving (localStorage)
- [x] Tutorial / first-level guide
- [x] Particle effects on death/completion
- [x] Screen shake on death
- [x] Death counter per level
- [x] Best attempts tracking
- [x] Sound effects via Web Audio API
- [x] Background music (procedural)

## Key Differentiators from Geometry Dash
- Name: "Neon Dash" (original, not trademark-infringing)
- Unique neon glow aesthetic
- Procedural BGM via Web Audio API
- Browser-optimized
- Free to play, no paywall
