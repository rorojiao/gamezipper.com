### Mechanic (spec for Phase 3 build)
- Player controls a lockpick (rotating rake/tension wrench) inside a lock cylinder.
- **Pin tumbler mechanic**: Align spring-loaded pins to their "shear line" by feel — each pin has a sweet spot you must find by rotating the pick and applying tension.
- **Core loop**: Apply tension (hold) → rotate pick through pin stack → find each pin's sweet spot (haptic/audio feedback: "click") → all pins set → lock opens → next level.
- **Visual**: Cross-section view of lock cylinder (like Skyrim/Fallout lockpicking UI). Pins shown as colored segments. Tension wrench at bottom.
- **Feedback**: Pin "click" sound + visual spring bounce. Wrong position = pin springs back (reset).
- **Difficulty progression**:
  - T1 (Padlock): 3-pin locks, generous sweet spots, slow timing
  - T2 (Deadbolt): 5-pin locks, tighter timing, security pins introduced
  - T3 (Combination Safe): Dial-based — listen for clicks while rotating dial
  - T4 (Warded Lock): Path-tracing — guide pick through internal wards
  - T5 (Digital/Electronic): Pattern sequence + timing hybrid
- **Levels**: 30 handcrafted levels across 5 tiers (6 per tier).
- **Scoring**: Time-based 3-star system. Hints available (show sweet spot glow). Undo (reset current pin).
