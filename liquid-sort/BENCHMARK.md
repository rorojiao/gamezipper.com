# Liquid Sort — Competitive Benchmark

## Competitors Analyzed
1. **Water Sort Puzzle** (IEC Global) — 100M+ downloads, market leader
2. **SortPuz** (Dark Halo) — 50M+ downloads
3. **Liquid Sort Puzzle** (KitarGames) — 1,200+ handcrafted levels, most feature-rich

## Key Differentiation Opportunities (missing from ALL competitors)
- No daily challenge mode
- No seasonal events
- No social features
- No streak rewards
- No themed collections

## Must-Have Features (from competitors)
- Tap-to-pour mechanic (core loop)
- Undo / Extra tube power-ups
- 3-star rating system (KitarGames differentiator)
- ASMR pouring sounds (critical retention hook)
- Smooth liquid pouring animation
- Progressive difficulty (3 colors → 12+ colors)
- Offline play
- Handcrafted levels (not procedural)

## GameZipper Implementation Plan
- 50+ handcrafted levels (expandable)
- 3-star rating per level (target moves)
- Daily challenge mode (unique to GZ)
- Streak rewards
- Undo + Extra tube hints
- Web Audio ASMR pouring sounds
- Canvas liquid physics with smooth animation
- Dark neon GameZipper aesthetic
- Tutorial system
- Progress save with localStorage
