⭕ Satogaeri
Nikoli circle-movement puzzle. Move circles so each country has exactly one circle. Numbered circles must travel the exact distance shown.

How to Play

  • Drag a circle along a straight line (horizontal/vertical) to move it.
  • Numbered circles must travel exactly that many cells.
  • Plain circles can move any distance (including staying put).
  • Circles cannot cross other circles' paths.
  • Goal: each country (region) ends with exactly one circle.
