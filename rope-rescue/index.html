How to Play

Objective

Cut ropes to release the character and guide them into the green exit door. Avoid spikes and hazards!

Controls

Swipe or drag across any rope to cut it. The character swings and falls with realistic physics.

Stars

Use fewer cuts for more stars. Each level has a target number of cuts for 3 stars.

Tools

Use Undo to restore the last cut. Use Hint to see which rope to cut first. You have 3 free hints, watch an ad for more.
