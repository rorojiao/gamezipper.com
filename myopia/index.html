About Myopia

Myopia is a loop-drawing logic puzzle from Grandmaster Puzzles. Draw a single non-intersecting loop using horizontal and vertical segments between grid dots. Arrow clues in each cell point toward all directions where the nearest loop segment can be found.

How to Play

Click between adjacent dots to draw loop segments. Toggle between Draw and Erase modes. The loop must form a single continuous cycle with no crossings or branches. Each arrow clue must correctly indicate the nearest segment direction(s).

Tips

Start with cells that have arrows pointing in only one direction - these strongly constrain where the loop must run. Remember that arrows point to the CLOSEST segment, so a cell with arrows pointing both up and down means the nearest segment is equidistant in both directions.
