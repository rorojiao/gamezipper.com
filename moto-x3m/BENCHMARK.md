# Moto X3M — Competitive Benchmark

## Reference Games
1. **Moto X3M** (original) — CrazyGames #1 motorcycle, 25 levels, time trials, stunts
2. **Moto X3M Bike Race** — Mobile version, 100M+ downloads, obstacle courses
3. **Drive Mad** — Poki 300M+ plays, physics driving, obstacle courses

## Core Systems Required
- Physics motorcycle (2D side-view, gravity, rotation)
- Accelerate/brake + lean forward/backward controls
- Obstacle courses: ramps, loops, moving platforms, explosives
- Star rating: 3 stars < fast_time, 2 stars < medium_time, 1 star = complete
- Flip detection: front/back flips give time bonus
- Checkpoint system: respawn on crash
- Level select with star display
- Best times (localStorage)
- Tutorial overlay
- Sound effects (engine, crash, flip, level complete)
- Mobile touch controls
- Analytics + SEO

## Visual Style
- Dark neon GameZipper theme
- Colorful obstacles
- Particle effects on crash/complete
- Smooth 60fps
