## Selected Candidate: Spin Rogue — Slot Machine Roguelike (23/25)

### Market Data
- **Luck be a Landlord** (TrampolineTales): 95% positive Steam, mobile port 2024, 50+ synergistic symbols, established streamer/YouTube presence through 2024-2026
- **Balatro** (2024): 500K+ copies, BAFTA 2025 — proves roguelike-puzzle-on-familiar-mechanic is a massive 2024-2026 category
- **Peglin** (2022): peg-pinball roguelike, 50K+ reviews
- Slot keywords ("slot game", "spin game", "casino puzzle") are high-search-volume and have ZERO GZ coverage — entire genre is a SEO greenfield
- Core mechanic: 3x3 (3x4 in original) grid of symbols; spin shuffles; each symbol pays coins via its own rule; symbols synergize (e.g., Bee +1 per adjacent Flower); pay escalating rent each floor; build synergistic set between floors via shop
- Audience: roguelike/casual/puzzle crossover — exactly GZ's core demographic

### Why Selected
- **Highest score (23/25)** of any R26 candidate
- **Massive genuine gap**: zero slot/symbol-machine/roguelike-slot on GZ (grep confirmed)
- **Proven demand**: 3 reference titles all 50K+ reviews, all still charting 2024-2026
- **Deep retention**: 30+ symbols × procedural synergy interactions = endless runs; escalating rent = inherent difficulty curve
- **SEO greenfield**: "slot roguelike" / "spin game" / "symbol puzzle" — high volume, zero GZ coverage
- **Feasibility**: slot reel animation + symbol effect engine + shop + rent escalation — well-understood single-file Canvas pattern

### Scoring
- D (Demand): 4 — 3 reference titles all proven, but slot-roguelike is subgenre not chart-topper
- S (SEO): 5 — entire slot-puzzle genre uncovered on GZ, high search volume
- R (Retention): 5 — roguelike depth, endless synergies, daily challenge potential
- F (Feasibility): 4 — symbol effect engine + reel + shop is medium-complexity single-file Canvas
- Z (Zero-overlap): 5 — strict grep = 0 slot/symbol-machine/roguelike-slot
