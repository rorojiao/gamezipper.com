## Selected Candidate: Metro Lines (24/25)

### Market Data
- **Mini Metro** (Dinosaur Polo Club): BAFTA-nominated, IGF award winner
- Desktop version: 250,000+ copies sold
- Mobile: millions of downloads, still charting 2026
- Genre: real-time subway/transit simulation puzzle
- **Core mechanic**: Draw subway lines connecting stations (shapes: circle, triangle, square, star, diamond, pentagon). Trains carry passengers (also shapes) to matching stations. Stations overcrowd if passengers queue too long → game over. Limited lines/trains/carriages. Weekly upgrades.

### Competitor Benchmarks
- **Mini Metro** (Dinosaur Polo Club) — the genre-defining hit
- **Mini Motorways** — roads variant by same studio
- **Transit King Tycoon** — transport management

### Scoring Criteria
- D (Demand): Downloads/rankings (5=BAFTA + 250K+ desktop + millions mobile)
- S (SEO): "mini metro", "subway game", "metro puzzle" — zero GZ coverage
- R (Retention): endless procedural cities, daily challenges, extreme replayability
- F (Feasibility): Canvas real-time sim with pathfinding — harder but achievable
- Z (Zero-overlap): grep confirmed no metro/transit/rail game
