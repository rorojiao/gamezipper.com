## Selected Candidate: Hotel Rush — Hotel Management Time-Management Puzzle (24/25)

### Market Data
- **My Perfect Hotel** (SayGames, 2023-2025): 100M+ downloads on Google Play, top-10 casual chart for 18+ months. Viral TikTok/YouTube gameplay. Core loop: assign guests → clean rooms → collect cash → build new floors → hire staff → automate.
- **Idle Hotel Empire / Hotel Empire Tycoon**: 50M+ downloads, idle-hotel genre proven.
- **Papa's Freezeria** lineage (already on GZ): proves time-management demand on web.
- **Hotel management is the #1 uncovered time-management category on GZ.**

### Why Selected (24/25 — highest of any R32 candidate)
- **D (Demand) 5**: My Perfect Hotel = 100M+ downloads, still top-10 casual in 2025-2026. Strongest demand signal of any gap.
- **S (SEO) 5**: "hotel game", "hotel management game", "my perfect hotel" — high search volume, zero GZ coverage.
- **R (Retention) 5**: Idle income loop + floor progression + staff upgrades = deep long-term engagement. Players return daily.
- **F (Feasibility) 4**: Top-down hotel view, guest entities, room grid, queue management. Moderate complexity but very achievable in single-file Canvas.
- **Z (Zero-overlap) 5**: Strict grep = 0 hotel/reception/check-in across 356 games.

### Gameplay Design
- Top-down/isometric hotel with multiple floors (start with 1 floor, expand to 5+)
- Guests spawn at entrance lobby, wait in queue (patience timer)
- Player taps guest → auto-walks to assigned empty room (or drag-assign)
- Guest stays for a duration, then leaves → room becomes dirty
- Player taps dirty room → maid cleans (takes time) → room ready again
- Earn cash per guest checkout
- Spend cash: build new floor, add rooms per floor, hire receptionist (auto-assign), hire maid (auto-clean), upgrade speed
- Idle income: earn offline cash based on hotel level (cap to prevent abuse)
- Daily reward / star rating per level milestone
- Difficulty curve: more guests, shorter patience, more floors, VIP guests (pay more, less patient)

## Selected Candidate: Mahjong Connect — Shisen-Sho Path-Drawing Tile Puzzle (25/25)

### Market Data
- **Mahjong Connect / Shisen-Sho**: Classic Asian tile puzzle, 100M+ cumulative downloads across mobile variants (Mahjong Connect, Onet Classic, Pokemon-style tile link games).
- **Onet / Pokemon Connect**: Indonesian-origin variant with symbol/fruit tiles instead of Chinese characters; massive in Southeast Asia (50M+ downloads).
- **Mahjong Chain**: Big Fish Games / GameHouse staple, top-10 portal puzzle for 15+ years.
- **Tile Connect / Link Link**: Modern mobile hits (Tile Connect - Onet Classic, Tile Master - Match Puzzle).
- Web portals (Poki, CrazyGames, Arkadium): Mahjong Connect consistently top-50 puzzle category, very high engagement.

### Why Selected (25/25 = perfect score)
- **Demand (5)**: 100M+ downloads reference; proven portal top-seller for 15+ years
- **SEO (5)**: "mahjong connect", "shisen sho", "shisen-seki", "tile connect", "onet classic", "mahjong chain", "link link" — all extremely high volume, ZERO GZ coverage
- **Retention (5)**: 1000s of layouts possible; classic + time attack + daily challenge + endless modes
- **Feasibility (5)**: 2D grid + BFS pathfinding (max 2 turns) + tile matching = clean single-file Canvas
- **Zero-overlap (5)**: Strict grep = 0 shisen/connect/liankan/onet; distinct from existing Mahjong Solitaire (pair-free matching) and Mahjong Dimensions (3D)

### Gameplay Design
- Grid board (typically 12x7 to 16x9 scaling with difficulty)
- 144-tile set (traditional Mahjong) OR symbol/fruit set (Onet variant)
- Click 2 matching tiles → if connectable by a path with ≤2 right-angle turns, both removed
- Path drawn visually as glowing line for satisfying feedback
- All tiles cleared → level complete
- Difficulty curve: more tile types, larger grids, time limits, special tiles (locked, frozen, multi-layer)
- Modes: Classic (untimed), Time Attack (beat the clock), Daily Challenge (seeded)
