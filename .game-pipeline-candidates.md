### Look-Air (#602) — Nikoli Rukkuea Logic Puzzle
- Benchmark: Nikoli Rukkuea (Look-Air) by cross-plus-a.com verified.
- Catalog gap: `look-air`, `look air puzzle`, `rukku`, `rukkuea` all 0 (grep verified).
- Core mechanic: shade square black blocks (1×1, 2×2, 3×3). Blocks must NOT touch orthogonally (only diagonally). Same-size blocks must NOT share an unobstructed row or column line of sight — a different-size block must interrupt. Each numbered clue = count of black cells in 5-cell window (self + 4 orth neighbors).
- 30 levels across 5 tiers: 5×5 Beginner → 9×9 Expert.

- Systems: click to place 1×1, click size selector for 2×2/3×3, right-click/erase mode, 3 hints/level, restart, 3-star ratings, level select, keyboard (1/2/3/E/H/R/C/Esc), Web Audio music (purple ambient chord progression), localStorage save, win overlay, settings (music/sfx/autocheck).
