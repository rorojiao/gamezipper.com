## Game Details

### 1. Meowdoku (23/25) — Tier 1
- Slug: meowdoku
- Gameplay: Sudoku + Minesweeper hybrid with cat theme. Place cats on a grid using row/column uniqueness constraints AND neighbor-count clues. Pure logic, no guessing. Daily puzzles.
- Why hot: #1 on iOS App Store puzzle chart (June 2026). Cat theme = massive viral appeal. Daily puzzle = high retention.
- Implementation: Canvas grid, two constraint types (uniqueness + neighbor counting), puzzle generation algorithm, daily seed.

### 2. Sand Sort Puzzle (23/25) — Tier 2
- Slug: sand-sort
- Gameplay: Pour colored sand between bottles to sort by color. Sand particles flow and mix realistically.
- Why hot: Widely cloned on browser portals. Sand physics distinct from liquid sort.
- Implementation: Particle-based sand simulation, bottle containers, pour mechanics.

### 3. Solitaire Roguelite (22/25) — Tier 2
- Slug: solitaire-roguelite
- Gameplay: Classic solitaire with roguelike progression — earn gold, buy power-ups, face escalating difficulty runs.
- Why hot: Genre crossover attracts card + roguelike audiences. High retention from run-based design.
- Implementation: Solitaire card engine + run tracker + upgrade shop.

### 4. A Little to the Left / Tidy Organize (22/25) — Tier 2
- Slug: tidy-organize
- Gameplay: Arrange everyday objects into satisfying order. Hidden sorting rules (size, color, type) must be deduced.
- Why hot: Spawned "cozy organization" sub-genre. ASMR/satisfying trend.
- Implementation: Drag-and-drop Canvas objects, rule validation, multiple level types.

### 5. Knotwords (21/25) — Tier 3
- Slug: knotwords
- Gameplay: Sudoku-like constraints + crossword word filling. Each cell has letter constraints.
- Implementation: Grid, constraint satisfaction engine, word dictionary, daily seed.

### 6. Draw Bridge Puzzle (21/25) — Tier 3
- Slug: draw-bridge
- Gameplay: Draw lines to create bridges, guide characters from A to B. Physics-based drawing.
- Implementation: Canvas line drawing, simple physics, character movement.

### 7. Gobble (20/25) — Tier 3
- Slug: gobble
- Gameplay: Control a monster that eats everything except humans. Strategic eating order puzzle.
- Implementation: Top-down Canvas, character movement, collision, eat animation.

### 8. Block Out! (19/25) — Tier 3
- Slug: block-out
- Gameplay: Sort colored blocks into containers with dynamic elevators and generators. 3D isometric.
- Implementation: Isometric Canvas, block physics, level scripting.

### 9. Blumgi Slime (18/25) — Tier 3
- Slug: blumgi-slime
- Gameplay: Control a squishy slime blob through physics puzzle levels.
- Implementation: Simplified spring-mass physics, Canvas rendering, level design.
