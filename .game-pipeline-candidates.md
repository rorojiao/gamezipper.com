1. **Draw to Save (draw a protective barrier to shield from danger)** — 22/25 ✅ SELECTED
   - Market gap 9: The "Draw to Save" / "Save the Doge" / "Protect My Dog"
     genre exploded to 100M+ downloads in 2022-2023 across multiple publishers
     (Wonderful Week, Gongay, SayGames). Mechanic: a cute character is
     threatened (bees swarm, spikes fall, lava rises, cannonballs fly); the
     player draws a single continuous line that becomes a solid barrier,
     shielding the character for N seconds (typically 10s). Genuinely uncovered
     on GZ: all existing draw games use DIFFERENT verbs (path-connect, physics-
     shape, water-guide, picture-complete, erase). Grep for save-doge, draw-
     protect, draw-to-save, line-shield, barrier-draw = ALL ZERO.
   - Search volume 4: "draw to save", "save the doge", "draw to protect",
     "protect my dog" — strong evergreen terms, the Doge meme alone drives
     millions of searches.
   - Monetization 5: Ultra-short satisfying sessions (draw 5s + watch 10s),
     perfect for interstitial between levels + rewarded "skip"/"hint" ads.
     Level-based, high binge replay. Top titles monetize at industry-leading
     eCPM.
   - Feasibility 2: Canvas freehand path drawing + polygon collision detection
     (threats move along predictable trajectories; check intersection with the
     drawn barrier polygon). Moderate: the collision math is the bulk of the
     work, but trajectories are simple (straight fall, radial swarm, linear
     rise). Single-file friendly.
   - Uniqueness 2: The "draw a barrier that physically blocks threats for a
     time limit" loop is mechanically distinct from ALL existing draw games.
   - Slug: draw-to-save · Tier 1 · cat: puzzle · emoji: 🛡️
   - Competitor features to match: freehand single-stroke drawing (finger/mouse),
     30 handcrafted levels (6 tiers × 5) escalating from "block falling spikes"
     → "shield from bee swarm" → "contain rising lava" → "deflect cannonballs"
     → "multi-direction threats" → "precision minimal-ink challenges", star
     ratings by ink efficiency (less ink = more stars), hints (ghost-suggest
     barrier shape), undo/clear-redraw, level select with progress save
     (localStorage), achievements, sound + procedural music, mobile-first touch
     + desktop mouse.
