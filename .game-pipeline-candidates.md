1. **Portal Bounce (bouncing ball + portal placement puzzle)** — 23/25 ✅ SELECTED
   - Market gap 9: `grep -c 'portal' games-data.js` = 0 across 427 games. The
     portal/teleport placement mechanic is COMPLETELY uncovered. Closest siblings
     (Inertia = ice-slide no portals; Light Beam Puzzle = mirror reflection not
     teleport; Pool = ball bounce no portals) are all mechanically distinct. The
     bouncing-ball + portal-placement combination is genuinely novel.
   - Search volume 3: "portal game", "portal puzzle", "teleport puzzle", "warp ball"
     are solid evergreen terms. Long-tail SEO ("portal bounce puzzle free online")
     has low competition.
   - Monetization 5: Level-based, ultra-short satisfying sessions (plan→launch→result
     in 10-30s), "aha!" moment per level, ideal for interstitial between levels +
     rewarded "show solution" / "skip" ads.
   - Feasibility 4: Grid-based Canvas (9x9 to 10x10), deterministic bounce physics
     (straight line + 90° wall reflection), portal teleportation (overlap detection),
     path pre-computation for animation + loop detection. Moderate: 30 levels must be
     hand-designed and verified solvable. Single-file friendly.
   - Uniqueness 2: Portal + bounce combination is mechanically distinct from any
     existing GZ game. Not a clone of Inertia, Light Beam, Pool, or Roll the Ball.
   - Slug: portal-bounce · Tier 1 · cat: puzzle · emoji: 🔮
