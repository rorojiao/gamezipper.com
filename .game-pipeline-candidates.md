## Scoring Criteria
- D (Demand): Downloads/rankings (5=100M+ downloads)
- S (SEO): Search volume × GZ zero coverage
- R (Retention): Player session length (longer = more ad revenue)
- F (Feasibility): Single-file HTML5 Canvas achievable
- Z (Zero-overlap): No similar core mechanic on GZ

## Candidate Details

### 1. Color Block Jam (24/25) — SELECTED
- **Mobile data**: 34M iOS downloads 2025, top-10 trending
- **Mechanic**: Slide colored blocks on parallel tracks to matching color gates. Traffic-jam style puzzle.
- **Why**: Pure logic puzzle, addictive "one more level" loop, trivial HTML5 Canvas implementation (2D grid, swipe), zero overlap (not tube-sort, not match-3), huge trending momentum.
- **Comparable GZ games**: Magic Sort (tube sort — different mechanic), Color Cascade (color matching — different mechanic)
- **Core loop**: Slide block → match gate → clear level → next puzzle. Simple but satisfying.

### 2. Tomb of the Mask (23/25)
- **Mobile data**: 498M all-time downloads
- **Mechanic**: Swipe to move character through maze, collect dots, avoid obstacles. Fast-paced arcade-puzzle.
- **Why**: Massive proven demand, swipe mechanic is HTML5-friendly, we have maze games but not this specific fast-swipe arcade style.

### 3. Goods Sort Challenge (22/25)
- **Mobile data**: 29M iOS downloads 2025
- **Mechanic**: Sort 3D goods on shelves by category/color. ASMR-style satisfaction.
- **Why**: Trending "sort" subgenre, different from tube-sort (shelf-based), satisfying visual feedback.
