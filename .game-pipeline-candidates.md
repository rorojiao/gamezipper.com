## Selected Candidate: Triple Match 3D Objects (22/25)

### Market Data
- **Trending subgenre**: "Match 3D" / "Goods Triple Match 3D" games dominate TikTok virality in 2025-2026
- **Google Play**: "Goods Triple Match 3D" — top trending puzzle, 10M+ downloads
- **Similar hits**: "Match 3D" (100M+), "Goods Master", "Triple Match 3D Shop"
- **Core mechanic**: Find 3 identical 3D objects (fruits, bottles, toys) scattered among piles on screen. Tap to collect into a 7-slot tray. When 3 matching items fill the tray, they clear. Clear all to win. Tray full = game over.

### Why Selected
- Completely unique visual mechanic on GZ (3D objects vs flat tiles)
- Top trending puzzle subgenre globally (TikTok viral)
- 100M+ downloads in the genre
- Highly satisfying gameplay loop (find-match-clear)
- Excellent retention (progressive difficulty, unlimited levels)
- SEO potential: "match 3d", "triple match", "goods match", "3d match puzzle"
- Feasible in Canvas 2D with depth simulation (scaling, z-ordering, shadows)

### GZ Gap Analysis
- GZ has "Tile Triple Master" — mahjong-style flat tile matching (different mechanic)
- GZ has "Goods Sort" — sorting goods into categories (different mechanic)
- GZ has NO 3D object pile matching game
- This fills a major gap in the trendy "3D match" category
