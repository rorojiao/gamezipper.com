1. **Peg Blast (Peggle-style aim-and-bounce peg-clearing puzzle)** — 25/25 ✅ SELECTED
   - Market gap 8: Peggle is PopCap's flagship franchise (50M+ downloads; "Peggle
     Blast" was EA's top mobile title 2014-2020, still heavily searched/played).
     The aim-cannon + gravity-bounce-off-pegs + clear-target-orange-pegs + moving-
     score-bucket mechanic is COMPLETELY uncovered. Plinko Drop exists but is pure
     drop (no aiming skill); Peg Solitaire is the jump-remove board game; Marble
     Shooter is straight-line chain matching. None reproduce Peggle's signature
     skill-based ricochet. Genuinely uncovered, high-demand mechanic.
   - Search volume 4: "peggle", "peg game", "peg blast", "bounce ball game",
     "cannon ball pegs" — strong steady terms, evergreen demand.
   - Monetization 4: Level-based, short satisfying sessions, "clear all orange
     pegs" objective creates replay compulsion, perfect for interstitial between
     levels + rewarded ads (extra balls).
   - Feasibility 4: Canvas + ball physics (gravity + circle-peg reflection) +
     aiming trajectory preview + peg grid + 30 levels. Well-understood single-
     file build. Standard 2D physics, no per-level art assets (colored circles).
   - Zero-overlap 5: The aim + bounce-off-pegs + clear-target-pegs + moving-bucket
     mechanic is mechanically unique in the portfolio. IP-safe (generic physics,
     no copyrighted characters; name is generic "Peg Blast" not "Peggle").
   - Slug: peg-blast · Tier 1 · cat: puzzle · emoji: 🎯
   - Competitor features to match: aim cannon with trajectory preview arc, gravity
     ball physics with peg ricochet, blue (normal) + orange (target) + green (free-
     ball) + purple (score) + gray (wall) peg types, clear-all-orange-to-win, 10
     balls per level, moving score bucket at bottom (catch = free ball), score
     multiplier + style points (long shots), 30 handcrafted levels (6 tiers × 5),
     level select with star ratings, hints, achievements, sound + music, save
     progress.
