## Game Details

### 1. Letter Boxed (25/25)
- Slug: letter-boxed
- Gameplay: Form words by connecting letters arranged on 4 sides of a square box. Each word must start with the last letter of the previous word. Use all 12 letters to win.
- Why hot: NYT daily word game with huge engagement. Strong SEO ("Letter Boxed today", "Letter Boxed answers"). Word-list only asset needed.
- Implementation: Canvas letter-ring, drag-to-connect, dictionary validation, daily puzzle seed.

### 2. 4 Pics 1 Word (22/25)
- Slug: four-pics-one-word
- Gameplay: 4 emoji images shown, guess the one word they all have in common.
- Why hot: Billion-download mobile franchise. Emoji proxy solves image-asset problem.
- Implementation: Emoji grid + letter scramble + coin/hint system.

### 3. NYT Tiles / Zen (21/25)
- Slug: nyt-tiles
- Gameplay: Relaxing tile-matching; find and tap matching tile pairs to clear layers.
- Why hot: NYT's Zen tile-matching game. Strong retention (daily play).
- Implementation: Layered tile grid, match-2, themed tilesets via emoji/unicode.

### 4. Ripple Effect (21/25)
- Slug: ripple-effect
- Gameplay: Nikoli logic puzzle. Fill each region with numbers 1-N (where N = region size). Same number cannot appear in same row/column within distance N.
- Why hot: Zero overlap with GZ. Clean backtracking solver. Strong puzzler community.
- Implementation: Grid with regions, click-to-enter numbers, constraint checker.

### 5. LITS / Nuruomino (20/25)
- Slug: lits
- Gameplay: Shade tetromino shapes (L/I/T/S) in each region so no two same shapes share an edge. All shaded cells connected.
- Why hot: Zero overlap. Unique constraint-based tetromino puzzle.
- Implementation: Click-to-shade cells, shape detection, connectivity checker.

### 6. Hotaru Beam (20/25)
- Slug: hotaru-beam
- Gameplay: Place bulbs and draw beams from numbered cells. Beams cannot cross. All cells illuminated.
- Why hot: Nikoli classic, zero overlap with GZ's akari (different mechanic).
- Implementation: Grid click-to-place, beam line drawing, constraint validation.

### 7. Country Road (19/25)
- Slug: country-road
- Gameplay: Draw a single continuous loop through the grid. Loop visits each room exactly once. Number clues indicate cells in room on the loop.
- Implementation: Click edges to draw path, loop validation, room count checking.

### 8. Bag / Corral (19/25)
- Slug: bag-puzzle
- Gameplay: Draw a single loop. Numbered cells inside loop indicate how many cells are visible in straight lines from that cell.
- Implementation: Edge-clicking loop drawing, visibility counting.

### 9. Stained Glass (19/25)
- Slug: stained-glass
- Gameplay: Color cells in a grid so no two adjacent cells share the same color. Number clues restrict valid colors.
- Implementation: Click-to-color, adjacency constraint checker, backtracking solver.
