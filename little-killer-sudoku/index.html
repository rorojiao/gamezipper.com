Welcome to Little Killer Sudoku!

This is a Sudoku variant with an exciting twist: diagonal sum clues.

Standard Sudoku rules apply: fill each row, column, and box with digits 1-9 (or 1-6) exactly once.

Diagonal Sum Clues

Numbers shown outside the grid tell you the sum of all digits along that diagonal.

The top-left number is the sum of the main diagonal (top-left to bottom-right). The top-right number is the sum of the anti-diagonal (top-right to bottom-left).

Highlighted cells on the board show the diagonal paths.

Tips & Tricks

Use the diagonal sums to narrow down possibilities. For example, if a diagonal of 9 cells sums to 45, every digit 1-9 must appear on it.

Tap a cell to select it, then tap a number to place it. Use the notes mode for pencil marks. Tap the hint button if you get stuck!
