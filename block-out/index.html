HOW TO PLAY BLOCK OUT

Drop 3D blocks into a vertical well. Fill an entire horizontal layer to clear it. Clear multiple layers at once for huge bonuses!

KEYBOARD CONTROLS

← → Move X axis (left/right)

↑ ↓ Move Z axis (front/back)

Q E Rotate Y axis (left/right)

W S Rotate X axis (forward/back)

A D Rotate Z axis (tilt)

Space Hard drop   ↓ Soft drop

H Hint   C Hold   P Pause

MOBILE CONTROLS

Swipe on canvas to move the block. Tap to drop. Use the on-screen buttons to rotate and access hint/hold.

SCORING

Place blocks to score. Clear layers for 100 × layers² × level. Consecutive clears build up combo multipliers. Earn up to 3 stars per level.
