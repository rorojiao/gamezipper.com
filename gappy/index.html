How to Play

Rules

  • Fill each cell with a number from 1 to N
  • Each row must contain every number exactly once
  • Each column must contain every number exactly once
  • Same numbers CANNOT touch - not even diagonally!

Tap a cell, then tap a number. Use the X button to erase.

Conflicts are shown in red. Solve all cells to win!
