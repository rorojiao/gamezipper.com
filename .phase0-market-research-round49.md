## Selected Candidate: GOLD MINER (gold-miner)

### Why Gold Miner?
1. **Massive market demand**: Classic Flash game with 100M+ downloads on mobile. One of the most recognizable casual game brands worldwide.
2. **Genuine mechanical novelty**: Pendulum hook physics + timed grab + reel mechanic. NO similar game in the 557-game catalog.
3. **High score**: 23/25 (Market=5, Novelty=5, SEO=4, Browser=5, Gap=5)
4. **Perfect browser feasibility**: 2D Canvas physics engine, single-file HTML5, no complex dependencies.
5. **Confirmed gap**: No pendulum-hook, gold-nugget, or mining-grab game exists. Claw Master (/claw-machine/) is a vertical claw drop game — fundamentally different mechanic.

### Mechanic Details
- A hook/claw hangs from a boom at the top of the screen, swinging like a pendulum
- Player taps/clicks to launch the hook in its current swing direction
- Hook extends diagonally into the underground mine
- If hook hits gold/diamond/gem → grabs it, retracts, adds value to score
- If hook hits rock/dynamite → grabs heavy junk, slow retract, low value
- If hook hits nothing → retracts empty
- Each level has a target score and time limit
- Between levels: shop to buy upgrades (stronger hook, dynamite, faster reel, etc.)

### Differentiation from existing games
- /claw-machine/ (Claw Master): Claw drops vertically to grab prizes in a machine. Static positioning.
- /treasure-dig/ (Treasure Dig): Grid-based excavation puzzle. Turn-based logic.
- /gold-miner/ (NEW): Physics pendulum + real-time timing + resource management. Fundamentally different.
