Welcome to Odd-Even Sudoku

A quick look at the rules

Goal: Fill every cell so each row, column, and box contains every digit from 1 to N exactly once.

Odd cell Cells with an orange circle must hold an odd digit (1, 3, 5, 7, 9).

Even cell Cells with a cyan circle must hold an even digit (2, 4, 6, 8).

Tap a cell, then tap a digit on the number pad. Use Notes to pencil candidates, Hint to reveal a digit, Undo to step back, Erase to clear.

How to Play

Goal: Fill the grid so every row, column and box contains each digit 1 to N exactly once.

Odd rule Cells marked with an orange circle must contain an odd digit (1, 3, 5, 7, 9).

Even rule Cells marked with a cyan circle must contain an even digit (2, 4, 6, 8).
