How to Play Fillomino

1. Some cells have clue numbers. These cannot be changed.
2. Fill every empty cell with a number from 1 to 9.
3. Each number N must form one connected group of exactly N cells (touching up/down/left/right).
4. Two different groups with the same number must NOT share an edge.
5. Fill all cells correctly to win!
