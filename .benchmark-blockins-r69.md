# Phase 2: Competitive Benchmark — Blockins

## Competitor Analysis
**Game:** Blockins (by Robert Alvarez, Poki)
**Genre:** Puzzle Platformer
**Platform:** Browser (Poki.com)

### Core Mechanics
1. Multiple tetris-shaped blocks per level (1-4 cells each)
2. Gravity-based platformer movement (left/right/jump)
3. Switch between active block (X/Space)
4. Rotate active block
5. Stack blocks on top of each other to reach higher platforms
6. Only ONE block needs to reach the portal to complete level
7. Colors: White, Blue, Red blocks with different properties

### Controls
- Move: WASD / Arrow Keys
- Switch block: X / Space
- Rotate: Up arrow / specific key
- Restart: R

### Monetization (Poki version)
- Inter-level ads (Poki SDK)
- No IAP

### Our Differentiation
- 30 designed puzzle levels (5 difficulty tiers)
- Mobile-first touch controls (on-screen D-pad + action buttons)
- Star rating system (based on moves/time)
- Hint system
- Web Audio BGM + SFX
- Dark neon aesthetic
- localStorage progress save

### Level Design (30 levels)
- Tier 1 (L1-6): Single block, basic jumping
- Tier 2 (L7-12): Two blocks, basic stacking
- Tier 3 (L13-18): Three blocks, rotation + stacking
- Tier 4 (L19-24): Switch puzzles + moving platforms
- Tier 5 (L25-30): Complex multi-block puzzles
