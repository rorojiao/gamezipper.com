## Selected: TILE MASTER — BitMango classic triple-tile match puzzle (50M+ downloads) — score 20/25 ✅

- **Slug**: tile-master
- **Type**: Classic triple-tile-match puzzle with a 7-slot tray and shuffle/undo power-ups
- **Source**: BitMango "Tile Master — Classic Triple Match & Puzzle Game" — 50M+ downloads
  on Google Play. Top-10 tile puzzle for 4+ years. Branded franchise with strong SEO.
- **Mechanic**: A grid of stacked tiles (emoji/pattern themed). Player taps tiles to send
  them to a 7-slot tray at the bottom. When 3 identical tiles enter the tray, they auto-clear.
  Clear all tiles on the board to advance. The tray holds max 7 — if it fills without a
  triple-match, the level fails. Power-ups: UNDO (revert last tap), SHUFFLE (rearrange
  remaining board tiles), HINT (highlight a valid triple).

### Why this is mechanically DISTINCT (per Principle 6 — 差异化衍生)

| Existing sibling | Mechanic | Why Tile Master is different |
|------------------|----------|------------------------------|
| Tile Connect Classic (/tile-connect/) | Connect 2 identical tiles via a path with ≤2 bends (Onet-style) | Tile Master uses a 7-SLOT TRAY + triple-match (3 tiles), NOT pairwise path-connect |
| Triple Tile Match (/triple-tile/) | 3D tile tap with tray | Tile Master uses 2D STACKED layers (tiles can be buried under others, revealing as you clear top layer) — Triple Tile Match is flat single-layer |
| Tile Triple Master (/tile-triple-master/) | 3D tile match with tray | Tile Master is 2D stacked-layer (depth reveal mechanic), distinct from 3D-perspective |
| Triple Match 3D (/triple-match-3d/) | 3D objects, find 3 identical | Tile Master is themed emoji tiles in 2D stacked layers, not 3D objects |

The "stacked 2D layers + 7-slot tray + shuffle/undo/hint power-ups" combination is the
signature of the original BitMango Tile Master. While we have tray-mechanic siblings, none
implement the **stacked-layer reveal** (tiles hidden beneath others, exposed only when the
top tile is tapped) which is Tile Master's defining feature.
