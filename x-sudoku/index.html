Welcome to X-Sudoku

A quick look at the rules

Goal: Fill every cell so each row, column, box — and both diagonals — contain every digit from 1 to N exactly once.

The X The two main diagonals (top-left to bottom-right, and top-right to bottom-left) form the X. They each must also contain all digits 1 to N exactly once.

Tap a cell, then tap a digit on the number pad. Use Notes to pencil candidates, Hint to reveal a digit, Undo to step back, Erase to clear.

Every puzzle has a unique solution. Earn 3 stars with no mistakes and no hints within the time target.

How to Play

Goal: Fill the grid so every row, column, box — and both diagonals — contains each digit 1 to N exactly once.

X rule Both main diagonals form an X across the board. Each diagonal must contain every digit exactly once.

Tap a cell then tap a digit to place it. Toggle Notes ✎ to pencil candidates, Hint to reveal a correct cell, Undo to step back, Erase to clear.

Star targets: 3 stars = solve within the tier time limit with no mistakes and no hints. 2 stars = 1 mistake/hint OR slower time. 1 star = completed.

Every puzzle has exactly one solution, verified by a solver.
