## 🔥 Top candidate: WOODOKU (Sudoku × Block Puzzle)

**Mechanic:** Place 3 random polyomino shapes per turn onto a 9×9 grid. Clear FULL rows,
columns, AND 3×3 boxes (sudoku sub-grid) to score. Game ends when no shape fits.
Distinct from all existing block games:
- `/block-blast/` = number-merge (match 3+ same numbers)
- `/wood-block-puzzle/` = 10×10 line-clear only (no boxes)
- `/blockudoku/` = 9×9 grid BUT single shape per turn, line OR box clear

**Market signal:**
- Woodoku (PlaySimple Games): 50M+ downloads, top-10 US puzzle charts for years
- "Wood Block Puzzle" / "Block Blast" combined: 2亿+ downloads, #1 global Jan 2026
- Branded search "woodoku online / free / play" + "block puzzle sudoku"
- Mechanic is unique: 3-shapes-per-turn (1010-style) × 3×3-box-clear (sudoku)
  hybrid — distinct verb from both existing siblings
