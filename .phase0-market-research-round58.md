### Mechanic (spec for Phase 3 build)
- **Core loop**: Arrange command blocks in sequence → press "Run" → robot executes commands on grid → light up all tiles + reach exit = win
- **Command blocks**: Move Forward, Turn Left, Turn Right, Jump (over gap), Light Up (toggle tile)
- **Grid types**: Floor tiles, gap tiles (can only cross with Jump), goal/exit tiles
- **Light tiles**: Some tiles must be "lit" (activated) by the Light command — robot must be standing on it
- **Subroutines** (T3+): P1 and P2 reusable command sequences — enables complex solutions with limited slots
- **Resource constraint**: Each level has a max number of command slots per main program and per subroutine
- **Difficulty progression**:
  - T1 (Hello Robot): 4-6 slots, straight-line paths, no jumps, 1-2 light tiles
  - T2 (Turning Point): 8 slots, turns required, simple jumps, 2-3 light tiles
  - T3 (Smart Code): 10 slots + P1 subroutine, complex paths, gaps, 3-4 light tiles
  - T4 (Code Master): 12 slots + P1+P2 subroutines, loops implied through P1/P2 reuse, 4-5 light tiles
  - T5 (Robot Genius): 16 slots + P1+P2, tight constraints requiring elegant solutions, 5+ light tiles
- **Levels**: 30 handcrafted levels across 5 tiers (6 per tier)
- **Scoring**: 3-star system based on command efficiency (fewer commands = more stars)
- **Features**: Step-through mode (pause between commands), undo/reset, level select, localStorage progress save, hint system (show partial solution), tutorial overlay for first-time players
