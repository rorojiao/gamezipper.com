How to Play

Goal: Color every cell in the grid so that no two adjacent cells (sharing an edge) have the same color.

Number Clues: Some cells show a number. This tells you which color that cell must be painted (1 = first color, 2 = second, etc.).

Controls: Select a color from the palette at the bottom, then tap/click cells to paint them. Tap a painted cell again to erase it.

Adjacency: Only cells that share an edge (not just a corner) count as adjacent.

Tips: Start with cells that have number clues. Use the hint button if you get stuck. Fewer moves and faster times earn more stars!
