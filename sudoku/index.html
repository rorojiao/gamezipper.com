Sudoku Online — Free Classic Number Puzzle

Play Sudoku for free online. Choose from three difficulty levels — Easy, Medium, and Hard. Fill the classic 9×9 grid using logic: each row, column, and 3×3 block must contain the digits 1 through 9 with no repeats.

How to Play

  • Select a difficulty level to start a new puzzle.

  • Enable Notes mode to mark candidate numbers in cells.

Frequently Asked Questions

How do you play Sudoku?

Fill the 9×9 grid so that every row, every column, and each 3×3 subgrid contains the digits 1 through 9. No number can repeat in the same row, column, or block.

What is notes mode in Sudoku?

Notes mode lets you pencil in small candidate numbers in empty cells. Toggle notes mode with the Notes button or N key. When you place a real number, it is automatically removed from notes in the same row, column, and block.

Is there a timer in this Sudoku game?

Yes! A timer tracks how long it takes you to solve the puzzle. Your best time is saved for each difficulty level.

What does error highlighting mean?

When error highlighting is on, any cell you fill that conflicts with Sudoku rules will be shown in red so you can quickly spot and correct mistakes.

Can I play Sudoku on mobile?

Yes! This Sudoku game is fully responsive and supports touch controls. Tap a cell to select it, then tap a number button to fill it in.
