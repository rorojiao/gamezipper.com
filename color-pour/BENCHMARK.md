# BENCHMARK.md — Color Pour Puzzle

## Competitor Analysis

### 1. Water Sort Puzzle (by Super App Studio / IEC Global)
- **Downloads**: 500M+ (Google Play)
- **Revenue**: $50K+/day from ads (hybrid monetization)
- **Mechanic**: Tap to pour colored water between tubes. Top segment of source must match top of target. Tube fills until source empty or target full.
- **Features**: 1000+ levels, hints, daily challenge, themes
- **Monetization**: Interstitial ads (every 3 levels), banner, rewarded video (hints)
- **Key Insight**: The pour mechanic (not swap) is what makes this viral — watching liquid transfer is ASMR satisfying

### 2. Color Sort Puzzle (by Zepni / Good Job Games)
- **Downloads**: 50M+
- **Mechanic**: Similar pour-based color sorting
- **Features**: 500+ levels, 3-star ratings, undo
- **Monetization**: Banner + interstitial

### 3. Pour Game / Liquid Sort variants (TikTok viral 2025)
- Multiple indie clones trending on TikTok
- Focus on ASMR visuals: realistic liquid sloshing, glass refraction, bubble sounds
- Simplified UI: no menus, just containers

## Differentiation Strategy

Our `color-pour` will differentiate from existing `water-sort`, `ball-sort`, `sand-sort`, `nut-sort`, `color-sort` by:

1. **True pour mechanic**: Unlike water-sort (which checks top segment match then swaps), our pour game simulates actual liquid volume — you can pour partially, creating layered colors in a single tube. This is more realistic and visually satisfying.
2. **ASMR liquid animation**: Wavy surface, pour stream, bubble effects, glass tube transparency
3. **Color mixing twist (higher tiers)**: Primary colors mix when poured together (red+blue=purple), adding a puzzle dimension beyond pure sorting
4. **30 handcrafted levels**: Progressive difficulty across 5 tiers
5. **Procedural background**: Colorful gradient backgrounds matching liquid theme
