## Core Mechanics
1. Each level shows a picture with a missing element
2. A text prompt describes what's missing (e.g., "Draw a tail for the cat")
3. Player draws with finger/stylus on the touchscreen
4. AI detects if the drawing is in the correct area
5. On success, the complete picture is revealed
6. Hint system shows the target area
