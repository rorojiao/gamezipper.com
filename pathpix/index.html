How to Play

🎯 Goal

Connect pairs of matching numbers by drawing paths of exact length. Each number tells you how many cells that path must cover. Complete all pairs to reveal a hidden picture!

📋 Rules

  1. Each number appears exactly twice on the grid — these are your endpoints.
  2. The number IS the required path length — draw a path of exactly that many cells.
  3. Paths can only go through adjacent cells (up, down, left, right).
  4. Paths cannot cross or overlap.
  5. When all pairs are connected, the filled cells reveal a hidden picture!

✨ Tips

  • The number on each pair tells you the exact cell count — count carefully!
  • Start with shorter paths first — they constrain the longer ones.
  • If stuck, use the 👁️ button to peek at the hidden picture.
  • Earn ⭐⭐⭐ by solving without hints and with fewer resets.
