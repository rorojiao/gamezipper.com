# Phase 0 Market Research — Round 79 (2026-07-08)

## Selected Candidate: Pulse Path (#589)

Benchmark family: ZHED-style exact-length relay path puzzle.

Evidence from web research:
- ZHED is distributed on Steam and Nintendo Switch.
- Steam/OurPlay snippets describe the mechanic: tap a square, choose a direction, see how squares react/intersect, build a path to fill the goal square.
- Listings describe 100 handcrafted levels, no timers/clocks, pure brain-teaser focus.

Catalog gap verification (GameZipper 588 live before build):
- `grep -ci zhed js/games-data.js` = 0
- `goal square`, `tap square direction`, `extend path` mechanic greps = 0
- Existing Numberlink/Flow/PathPix games draw/free-route paths; none uses exact-length relay jumps from numbered nodes.

Score: 23/25 (market 8, SEO gap 5, monetization 4, feasibility 3, uniqueness 3).

Outcome: built Pulse Path as a differentiated neon relay puzzle with 30 unique levels, exact-length pulses, decoy relays, blockers, hints, undo, stars, Web Audio, and procedural art.
