How to Play

🎯 Goal: Drag the nodes (circles) so that no wires (lines) cross each other.

🟢 Green wire = no crossings

🔴 Red wire = still crossing another wire

🖱 Desktop: Click and drag any node.

📱 Mobile: Touch and drag any node.

↶ Undo reverses the last move.

↻ Reset returns to the scrambled start.

💡 Hint highlights a useful move.

⭐ Stars: 3 stars = minimal moves, 2 = moderate, 1 = completed.

Every level is generated from a planar graph — a solution is always guaranteed!
