Build a complete, commercial-quality HTML5 Bloxorz (Block Roll Puzzle) game.

## Game Description
A rectangular 1x1x2 block rolls on a grid of floating tiles. The goal is to navigate the block to fall into the exit hole by rolling it across the grid. The block can be flat (covering 2 tiles), standing (covering 1 tile vertically), or split into two 1x1 cubes. The game features switches, bridges, teleporters, and fragile tiles across 33 hand-crafted levels with increasing difficulty.

MUST implement ALL mechanics from the original Bloxorz plus modern features:

- Special tiles: soft switch (any orientation), heavy switch (standing only), split switch (teleport+split), bridge tiles (toggle via switches), fragile tiles (break if standing), exit hole
- Split mechanic: spacebar toggles between split cubes, cubes rejoin when placed adjacent
- 33 hand-crafted levels with passcodes (6-digit level codes for level select)
- Move counter per level
- Par scores for each level (minimum moves)
- Star rating system: 1 star (completed), 2 stars (under 2x par), 3 stars (under par)
- Level select screen with star tracking
- Undo system (unlimited undo)
- Hint system (shows next optimal move direction)
- Tutorial for first 3 levels
- Progress save (localStorage with version field)
- Keyboard (arrow keys + spacebar) AND touch/swipe controls for mobile
- Responsive design for desktop and mobile
