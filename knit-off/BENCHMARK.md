# Knit Out — Competitive Benchmark

## Original Game
- **Developer**: SayGames / Rollic Games
- **Platform**: Mobile (iOS + Android)
- **Rating**: 4.5+ stars
- **Downloads**: 10M+
- **Levels**: 1000+

## Core Mechanic
- Thread/bobbin color matching puzzle
- Colorful yarn knots appear at top of screen
- Player taps knots to collect into waiting area (limited slots)
- Match yarn colors to corresponding bobbins below to clear fabrics
- If waiting area fills up → level failed
- Strategic planning: every move counts

## Difficulty Progression
- Levels 1-20: Easy tutorial, 2-3 colors
- Levels 21+: More colors, more knots, obstacles
- Layered bobbins (multi-layer) feature in later levels

## Key Features
- No timers (relaxed gameplay)
- Boosters: Scissors, Bobbins Up, Shuffle
- Obstacles: Locks, frozen bobbins, hidden fabrics
- Visual feedback (red when 1 slot left)
- Clean tutorial levels
