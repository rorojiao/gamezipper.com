# Tower of Hanoi Game Specification

## Game Overview
- Slug: tower-of-hanoi
- Display Name: Tower of Hanoi
- Category: puzzle
- Tags: classic, logic, recursion
- Level Count: 27 (6 tiers)
  - Tier 1: 3 disks, 3-rod (5 levels)
  - Tier 2: 4 disks, 3-rod (5 levels)
  - Tier 3: 5 disks, 3-rod (5 levels)
  - Tier 4: 5 disks, 4-rod (5 levels)
  - Tier 5: 6 disks, 3-rod (4 levels)
  - Tier 6: 6 disks, 4-rod (3 levels)

## Core Mechanics
- Objective: Move all disks from source rod to target rod
- Rules:
  - Only one disk can be moved at a time
  - A disk can only be placed on a larger disk or empty rod
  - Minimize number of moves
- Scoring:
  - Base score: 1000 per level
  - Bonus for optimal moves: 50 per move saved
  - 3-star thresholds based on move counts

## Input & Controls
- Pointer events for touch/mouse
- Drag-and-drop disks between rods
- Tap to select, tap to move
