## Selected: Tap Away

**Mechanic:** 2D grid of blocks. Each block has a directional arrow (N/S/E/W). Tap a block to slide it off the grid in its arrow direction. A block can only move if its path is clear. Remove all blocks to win.

**Score: 23/25**
- Search Volume: 5/5 (100M+ downloads, top puzzle charts worldwide)
- Uniqueness: 5/5 (zero tap-away or 3D block-removal games on GZ)
- Build Feasibility: 4/5 (2D grid logic, Canvas, clean physics)
- Engagement: 4/5 (satisfying, progression-based, replayable)
- Mobile: 5/5 (tap-based, perfect for touch)

**Slug:** `tap-away`
**Tier plan:** T1 (3x3, 4 blocks) → T5 (7x7, 30+ blocks), 30 levels total
