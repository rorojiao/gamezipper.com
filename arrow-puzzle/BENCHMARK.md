## 2. Core Mechanic (chosen)

**Spin-to-Align (Lights-Out variant with arrows):**
- Grid of N×N arrow tiles, each pointing one of 4 directions (↑↓←→).
- Tap a tile → rotates THAT tile + its 4 orthogonal neighbors 90° clockwise.
- Goal: make ALL tiles point to the TARGET direction (default UP ↑).
- Win when entire grid is aligned to target.

This mechanic is:
- Instantly graspable (one rule)
- Mathematically deep (mod-4 linear algebra over Z4 — solvable by design)
- Proven addictive (Lights-Out lineage, 100M+ lifetime downloads across variants)

## 6. Numerical Design
- **Star thresholds**: 3★ = par moves, 2★ = par×1.5, 1★ = par×2
- **Hint cost**: reveals optimal next move (greedy heuristic)
- **Combo**: consecutive taps without undo = small score multiplier (cosmetic)

## 7. Level Solvability Guarantee
Every level is generated by:
1. Start from SOLVED state (all arrows = target).
2. Apply K random valid taps (K = difficulty factor).
3. Resulting state is GUARANTEED solvable in ≤ K moves (reverse the taps).
4. Par = K (achievable). Stored par drives star thresholds.

This guarantees 100% solvable levels with known optimal solutions.
