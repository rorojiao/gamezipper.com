Click between two cells to toggle a border. Each rectangle must contain exactly one clue where clue = width + height.
How to play: Divide the grid into rectangles. Click the edge between two cells to draw/erase a border. Each rectangle must contain exactly one clue, and the clue number must equal the rectangle's width + height (e.g., clue 5 = 2×3 or 1×4). Rectangles tile the grid with no overlaps or gaps. Keys: 1 draw, 2 erase, H hint, R clear, Enter check, Esc menu.
