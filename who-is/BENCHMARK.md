# Who Is? Detective Logic Puzzle - Competitive Benchmark

## Game Overview and Genre Positioning

**Game Title:** Who Is? (also known as "Who Is? Brain Teaser & Riddles")
**Developer:** Unico Studio
**Platforms:** Browser (Poki), iOS App Store, Google Play, Android
**Genre:** Detective Logic Puzzle / Brain Teaser / Hidden Object Riddle Game

### Market Position
- **Poki:** #4 most popular puzzle game, 4.1 stars (2,015,185 votes)
- **Google Play:** 4.6 stars (1.8M reviews), 50M+ downloads
- **Likes:** 1.6M | **Dislikes:** 457.7K (Poki)
- **Sequel:** Who Is? 2 Brain Puzzle & Chats (also on Poki and mobile stores)

### Core Description
"Who Is?" tests puzzle-solving skills through tricky brain-teasers featuring cartoon visuals. Players encounter logic puzzles and hidden object challenges where they must "spot the liar" and solve detective-style riddles. Each level presents unique scenarios requiring outside-the-box thinking.

## Core Mechanics Breakdown

### Primary Gameplay Loop
1. **View scenario** - Characters/scenes displayed with visual narrative
2. **Interact** - Click/tap objects to examine clues
3. **Deduce** - Analyze findings to determine "who did it"
4. **Answer** - Select correct character or provide solution
5. **Progress** - Advance to next level (or use hints if stuck)

### Interaction Types
| Interaction | Description |
|-------------|-------------|
| Tap/Click objects | Examine items for hidden clues |
| Drag objects | Move items to reveal hidden elements |
| Combine items | Use multiple objects together |
| Select character | Point to the suspect/answer |
| Text input | Type answers to riddles (occasional) |

### Puzzle Categories in Game
- **Who is lying?** - Identify the deceptive character
- **Who is the killer?** - Solve mini murder mysteries
- **Who is the imposter?** - Find the impostor among real participants
- **Hidden object** - Find concealed items in scenes
- **Riddle solving** - Decrypt text-based puzzles
- **Scene examination** - Analyze visuals for inconsistencies
- **Logical deduction** - Connect clues to form conclusions

### Hint System
- **Help button** at bottom of screen when stuck
- **Skip level** functionality available
- Hints reveal partial information without full solution

### Progression
- 100+ unique levels (Poki description: "over a hundred tricky scenarios")
- Progressive difficulty increase
- New puzzle types introduced as player advances
- Offline play supported (mobile)

## Level Design Patterns

### Pattern Types Observed

**1. Scene Investigation**
- Examine crime scene for evidence
- Tap objects to collect clues
- Find hidden items within illustrations

**2. Character Interrogation**
- View character statements
- Identify inconsistencies in testimony
- Spot the liar through careful observation

**3. Object Manipulation**
- Drag items to new positions
- Combine multiple objects
- Reveal hidden elements

**4. Riddle-Based**
- Text appears with puzzle
- Player must deduce answer
- Sometimes involves wordplay

**5. Visual Deduction**
- Compare multiple images/scenes
- Find differences
- Identify anomalies

### Difficulty Progression
- **Early levels (1-20):** Tutorial-style, obvious clues
- **Mid levels (21-60):** Requires more examination, multi-step deduction
- **Late levels (61-100+):** Complex scenarios, subtle clues, multiple suspects
