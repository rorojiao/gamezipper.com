Castle Wall

Draw a single route from Start to Goal. White arrows point to cells the path must cross; black arrows to cells it must avoid. 28 hand-crafted logic puzzles.

♚ How to Play

Goal

Connect the green Start flag to the gold Goal flag with a single continuous path.

Drawing

  • Tap the edge between two cells to draw a line segment. Tap again to erase.
  • The path cannot cross itself or branch off.
  • Start and Goal each have one connection; every other visited cell has exactly two.

Arrow Clues

Arrows sit on blocker cells the path cannot enter. Each arrow has a direction, a number (step distance) and a color:

↑ White = path MUST cross the target cell
↓ Black = path must AVOID the target cell

The number tells how many steps in the arrow's direction the target cell lies from the clue.

Tips

  • Use Check to test your route for clue violations.
  • Stuck? Use a Hint to reveal one correct segment (3 per level).
  • Undo steps back; Reset clears the board.
