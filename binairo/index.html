Binairo

The Binary Puzzle — fill the grid with 0s and 1s using pure logic

Choose a Level

The Four Rules

1. Fill every cell with 0 or 1.
2. No three identical digits in a row or column.
3. Each row and column has equal 0s and 1s.
4. No two rows are identical; no two columns are identical.

Spot Forced Moves

If you see 00 next to each other, the cells on both ends must be 1 — otherwise you get three in a row.

Tools & Stars

Select a cell, then tap 0 or 1. Use Undo to revert, Hint when stuck (3 per level). Earn 3 stars by solving fast with no errors and no hints. Conflicts glow red in real time.
