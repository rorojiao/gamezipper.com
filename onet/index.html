ONET CONNECT

TILE MATCHING PUZZLE

HOW TO PLAY

Match pairs of identical tiles by connecting them with a path.

Path Rules:

  • Path can have at most 2 bends (turns)
  • Path cannot pass through other tiles
  • Path can go outside the grid border

Power-ups:

  • Hint: Shows a valid pair (3 per level)
  • Shuffle: Rearranges tiles (2 per level)
  • Undo: Reverts last match (unlimited)

Scoring:

  • Base: 100 points per match
  • Combo: Consecutive matches within 3 seconds multiply score
  • Time Bonus: Remaining seconds x 10

Complete levels under time limit to earn stars. Faster completion = more stars!
