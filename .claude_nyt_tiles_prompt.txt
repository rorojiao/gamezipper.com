## Game Overview
NYT Tiles is a relaxing tile-matching puzzle game where players tap pairs of tiles that share visual elements to clear them. Key mechanic: select tile A, then select tile B - all visual elements common to BOTH tiles are cleared from both. Tiles do NOT need to be adjacent. Build chains to clear the entire board.

## Core Gameplay Mechanics
1. **4-Layer Grid System**: 45 tiles total (9+12+12+12), layered A1-A9 / B1-B12 / C1-C12 / D1-D12
2. **Shared Element Wipe**: Select tile A, then tile B → all matching elements cleared from both tiles
3. **Chain System**: Continue selecting pairs without breaking streak. Empty tiles don't break combo.
4. **Perfect Clear Bonus**: Clear entire board without breaking the chain
5. **Tile Elements**: Emoji/unicode symbols (4-6 elements per tile) in themed categories
6. **Daily Puzzle**: Date-seeded puzzle rotation (refreshes at 12am EST)
7. **Practice Mode**: 30+ practice levels with curated tile sets

## Critical Implementation Details
- **Shared Element Detection**: Use Set intersection - match = tileA.elements ∩ tileB.elements
- **Chain Validation**: Empty tiles in path don't break streak (only invalid selections break)
- **Tile Element Storage**: Array of emoji strings per tile, e.g., `['🌟', '🔥', '⭐']`
- **Seeded PRNG for Daily**: Use mulberry32 or similar with YYYY-MM-DD seed
- **Canvas Layers**: Draw tiles from bottom layer (D) to top (A) for correct z-order
- **Touch Handling**: Use pointerdown/pointerup for unified touch + mouse support
- **Performance**: Batch canvas redraws, use requestAnimationFrame with delta time
