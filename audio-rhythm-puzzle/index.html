How to Play

1. Listen

Tap "Play Sequence" to hear a short melody. Each note lights up its colored button in turn.

2. Repeat

Tap the same color buttons in the same order. Each tap must match the next note in the sequence.

3. Progress

Earn up to 3 stars per level. Unlock the next tier when you complete a level.

Tips

  • Use the Replay button if you forget the sequence
  • Each wrong tap costs a life — you start with 3
  • Hint reveals one note (costs 1 life)
  • Build a daily streak for bonus points
