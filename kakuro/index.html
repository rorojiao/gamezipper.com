HOW TO PLAY

Kakuro is a number puzzle where you fill empty cells with digits 1-9.

Each horizontal row of numbers must sum to the clue on its left.

Each vertical column must sum to the clue at its top.

No digit can repeat within the same sum line.

Use Notes mode to mark possible candidates.
