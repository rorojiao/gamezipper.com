## Features to Implement (S-Grade)

### Core
- Grid sizes: 3x3, 4x4, 5x5, 6x6, 7x7
- Timer with millisecond precision
- Best times per grid size (localStorage)
- Error counter (wrong clicks)
- Next number indicator

### Game Modes
- Numbers (ascending 1→n)
- Numbers (descending n→1)
- Letters (A→Z, for 5x5 and up)
- Colors (find colored cells in order)

### Stats & Progress
- Personal best per grid size + mode
- Session history (last 20)
- Average times
- Error rate tracking
- Streak tracking (consecutive days played)

### Visual
- Dark gradient background (GameZipper style)
- Smooth cell animations (pop on click, pulse on hover)
- Found-cell visual feedback (color change + scale)
- Center dot option for peripheral vision training
- Celebration particles on completion

### UX
- Tutorial (first-time user guide)
- Settings panel (sound, center dot, animations)
- Share results (copy text)
- Mobile-first: large tap targets
