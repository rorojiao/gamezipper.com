How to Play Nawabari

1Goal: Divide the grid into rectangles by drawing borders between cells. Every cell must belong to exactly one rectangle.
2One clue per rectangle: Each rectangle must contain exactly one numbered cell. No rectangle can have zero or multiple clues.
3Clue meaning: The number tells how many of the clue cell's four edges are borders (including the grid's outer edge). A 4 means the cell is an isolated 1x1. A 0 means no edges are borders.
4Drawing borders: Click on the edge between two cells to toggle a border line. The grid's outer edge is always a border. Borders must form rectangles only — no L-shapes.
5Win: All borders form valid rectangles, each with one clue, and all clue values match their border-edge counts. Use Hints if stuck!
