# Color by Number — 竞品对标文档

## 竞品分析

### 1. Happy Color (Easybrain)
- **下载量**: 100M+ (Google Play), #1 casual art game
- **核心机制**: 按数字填色，每个区域标有对应颜色编号
- **图片数量**: 10000+ 图片，持续更新
- **分类**: Animals, Nature, Food, Vehicles, Fantasy, Flowers, Mandala, People, Places, etc.
- **系统**:
  - 每日一画 (Daily puzzle)
  - 故事模式 (Stories - Gatsby等IP合作)
  - 收藏功能
  - 搜索功能
  - 自动完成提示
  - 分享完成作品
  - 进度保存
- **美术风格**: 高质量插画风格，色彩丰富
- **音乐**: 轻松舒缓的背景音乐

### 2. Pixel Art - Color by Number (Easybrain)
- **下载量**: 50M+
- **核心机制**: 像素风格 color by number
- **特点**: 
  - 像素网格 vs Happy Color 的区域填充
  - 3D 像素画效果
  - 像素动画效果
- **系统**:
  - 关卡难度递进（从简单到复杂像素图）
  - 成就系统
  - 每日挑战

### 3. Color Tap (CrazyGames)
- **平台**: Web browser
- **核心机制**: 点击填色，区域高亮
- **特点**:
  - 简单易上手
  - 放大/缩小功能
  - 颜色提示
  - 自动保存进度
- **适合Web**: 快速加载，单页体验

### 4. Color Pixels (Playgama)
- **平台**: Web browser
- **核心机制**: 像素网格 color by number
- **特点**:
  - 动物主题为主
  - 单次点击填色
  - 完成后展示完整画面

### 美术风格
- **参考**: Happy Color 的高质量插画风格
- **GZ实现**: Canvas 绘制的低多边形/矢量风格图片
- **颜色**: 丰富但和谐的调色板，每图10-20种颜色
- **背景**: 深色渐变背景（GZ 统一风格）

### 音乐风格
- **参考**: 轻松、治愈、Lo-fi
- **BGM**: Web Audio 程序化生成
- **音效**: 填色(柔和)、完成(庆祝)、提示(提示音)、错误(低沉)

## 关卡设计
- **数量**: 至少30张图
- **难度递进**: 
  - Easy (1-10): 简单形状，5-8种颜色
  - Medium (11-20): 中等复杂，10-15种颜色
  - Hard (21-30): 复杂图片，15-20种颜色
- **每图数据**: 区域多边形 + 颜色编号 + 正确颜色值
