How to Play

Queens is a logic puzzle
  • Each row needs exactly one crown
  • Each column needs exactly one crown
  • Each color region needs exactly one crown
  • Crowns cannot touch — not even diagonally
  • Tap a cell to cycle: empty → X → crown → empty
  • Every puzzle has exactly one unique solution
