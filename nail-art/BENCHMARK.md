# BENCHMARK — Nail Art Studio

## Top competitor analysis

### 1. Nail Salon Makeover Studio 3D (iOS/Android)
- **Platform**: iOS (App Store id1595777890), Android
- **Genre**: ASMR / craft / simulation
- **Core loop**: Choose a hand → file nails → apply base coat → paint color → add art/stickers → dry → match target or free-create
- **Features**: acrylic nails, gel polish, glitter, unicorn designs, rainbow, rhinestones, festive themes
- **Monetization**: Interstitial + rewarded video ads between steps
- **Audience**: Casual, primarily female, ASMR/satisfying seekers

### 3. Bubble POP IT ASMR (Steam) — adjacent category
- **Takeaway**: ASMR + craft is a proven premium category (paid Steam release)

## Our differentiation: PUZZLE, not sandbox
Competitor apps are **free-play sandboxes** — no fail state, no levels, no puzzle. Our game adds the **puzzle layer**:
- Each level has a **TARGET design** the player must recreate
- Tools must be applied in the **correct sequence** (base → color → pattern → sticker → top coat)
- **Star rating** based on accuracy + step efficiency
- **30 handcrafted levels** across 5 tiers (Beginner → Master)
- **Color mixing** puzzle element (red + white = pink, blue + yellow = green)
- **Daily challenge** + **8 achievements**

This matches our proven craft-puzzle template (Candle Crafter: pour wax layers in order; Pottery Master: sculpt to match silhouette; Gem Paint: place gems on numbered grid).
