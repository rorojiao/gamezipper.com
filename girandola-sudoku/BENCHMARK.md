# Girandola Sudoku — Competitor Benchmark

## What is Girandola Sudoku?

Girandola Sudoku adds one extra constraint to standard Sudoku: a fixed set
of N cells (the "Girandola region," shown in rose tint) must also contain
each digit 1..N exactly once — forming an extra region beyond rows, columns,
and boxes. On a 9×9 grid these are the 9 box-center cells arranged in a
symmetric pinwheel. On a 6×6 grid, 6 windmill cells serve the same role.
