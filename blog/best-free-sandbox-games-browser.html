Sandbox vs. Open-World: What's the Difference?

Sandbox games focus on player creativity and emergent gameplay — building, crafting, experimenting with no fixed goals. Open-world games focus on exploration within a pre-designed world — the map is fixed, but you can roam freely. The best games blend both: Minecraft is a sandbox with an open world, GTA V is an open world with sandbox elements, and Stardew Valley is a sandbox with light open-world farming.

Every game on this list is a true sandbox — you set the goals, the game gives you the tools.
