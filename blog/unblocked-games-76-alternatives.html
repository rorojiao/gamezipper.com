What Types of Games Work Best on School Networks?

Not all games work equally well on restricted networks. Here's what to look for:

  • Client-side only (HTML5) — games that run entirely in the browser without needing a game server
  • Small file size — under 1MB for the initial load means faster loading on slow school WiFi
