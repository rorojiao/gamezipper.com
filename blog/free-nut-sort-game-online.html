What Is Nut Sort?

Nut Sort is a sorting puzzle with a meta-progression layer. The core loop is familiar: bolts of mixed-color nuts, you drag the top nut from one bolt to another, only the top nut moves, only same-color stacking is allowed, sort all colors to complete the level. The twist is what happens after you complete a level — you earn resources, and those resources go into a city-building side game.

This is the same design philosophy that made games like Royal Match, Homescapes, and Candy Crush Soda Saga billion-dollar hits: pure puzzle gameplay wrapped in a progression system that makes every level feel like it's contributing to a larger goal.

How to Play Nut Sort

The basic controls are intuitive:

  1. Pick up a nut — Tap or click on a bolt to pick up the top nut.
  2. Move the nut — Drag the nut to another bolt. You can only place it on an empty bolt or on top of nuts of the same color.
  3. Sort all colors — Continue sorting until each bolt contains only one color.
  4. Earn stars — Use fewer moves to earn more stars. 3 stars = optimal solution.
  5. Spend rewards — Use coins and materials to build or upgrade city buildings.
  6. Daily challenge — A bonus level every day with extra rewards for completion.
