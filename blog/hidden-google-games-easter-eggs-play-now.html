🦖 Chrome Dino — The Most Famous Hidden Game

How to Play Chrome Dino

Hidden GameEndless RunnerOffline

You know that T-Rex game that appears when Chrome has no internet? That's Chrome Dino — and it might be the most played game in the world.

How to access it:

  • Open Chrome browser
  • Go to any URL (or just visit chrome://dino)
  • Press Space or Up Arrow to jump
  • Press Down Arrow to duck under pterodactyls

You can also play it right now with an internet connection — just search "chrome dino game" on Google!

Pro tip: The Chrome Dino game gets harder over time. Your high score is saved locally. The longest anyone has officially run the Chrome Dino game is over 900 days of real-time play — yes, someone actually left it running continuously.

🐍 Google Snake — Hidden in Search

Google Snake Game

Hidden GameClassicSearch

Google hid a fully playable Snake game right in Search. It's been there for years and most people never know it exists.

How to play:

  • Open Google Search (google.com)
  • Type "snake game" and hit search
  • The Snake game appears as a featured result at the top
  • Use arrow keys to move

It's a modern take on the classic Nokia Snake game, playable directly in your browser with smooth controls.

🤖 Android-Specific Hidden Games

Android Easter Egg

AndroidEaster EggHidden

Every Android version hides a game in its settings. The method changes each version:

  • Android 12/13: Settings → About Phone → Android Version → tap the logo repeatedly → a cat-themed game appears
  • Older versions: Settings → About Phone → Android Version → tap repeatedly → Flappy Bird-style game or a spinning logo

Chrome T-Rex Runner API

Hidden GameAPIDeveloper

Google actually exposed the Chrome Dino game as an open-source runner anyone can embed. Developers use it as a template for endless runner games. The source code is on GitHub and it's a great learning resource.

Frequently Asked Questions

Does Google have hidden games?

Yes! Google hides games all over its products — in Chrome (Chrome Dino), in Search (Atari Breakout, Solitaire), on Android, and more. This guide covers every hidden Google game we know about.

How do I play the Chrome Dino game?

Open Chrome, disconnect from the internet (or just visit chrome://dino), and press Space or Up Arrow to start. Jump over cacti, duck under pterodactyls. That's it — endless runner goodness.

What happens when you search 'snake game' on Google?

Searching 'snake game' on Google shows an interactive Google Snake game directly in the search results — a playable version of classic Snake built into Google Search.

Is Chrome Dino open source?

Partially! The game is based on an open-source endless runner concept, and there are many open-source clones of it on GitHub. Google has also exposed parts of the runner API for developers.
