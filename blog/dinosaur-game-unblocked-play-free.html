What Is the Dinosaur Game?

The dinosaur game (also called the T-Rex runner or Chrome Dino) is a browser game built into Google Chrome. It appears when you try to load a webpage with no internet connection. The original game was created in 2014 by the Chromium team as an Easter egg. It became so popular that Chrome added it permanently to the offline error page.
